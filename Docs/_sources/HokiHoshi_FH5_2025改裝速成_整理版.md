## 開場：通則

### 1. PI 拉滿

**永遠把車建到該級距的最高 PI**。任何低於上限的 PI 都是浪費的性能——別把分數留在桌上。

### 2. PI 不是 100% 準確

> **同 PI 的兩輛車不一定一樣快**。PI 系統是粗略的近似，不受調校影響，且**不同車在同一級距內天生有強弱**。

### 3. 升級限度

- 多數車在原廠級距或**升 1~2 個級距**內表現最好
- **過度升級低級距車**完全可能造成「難開、競爭力差」的 build
- **不要無腦把所有車都頂滿升級**

### 4. 規則總有例外

> 即使在 FH5 上市超過 3 年的今天，**頂尖玩家**仍持續實驗 build 與調校。你也應該如此實驗。

## 一、改裝順序：由下往上

從 Conversions 開始，往上做。

### 1.1 引擎置換（Engine Swap）

> **作者推薦的最強三組引擎置換**：
> - **6.2 L V8**
> - **Racing V12**
> - **2 L Inline-4 VVT**

若有其中之一可選且在 PI 預算內 → 強力推薦。**否則，預設保留原廠引擎**。

### 1.2 進氣置換（Aspiration Swap）

> **Centrifugal Supercharger（離心式機械增壓）是最強的一個**

- Turbo 升級也 OK，特別是需要跨 PI 跳級時
- 很多車**不裝增壓也夠用**（保留原廠 NA 或原廠增壓）
- 取決於 PI 預算與目標級距

### 1.3 驅動置換（Drivetrain Swap）

- **預設**：保留原廠驅動（特別是新手）
- 但部分車的 **AWD 置換是好升級**
- **FWD**：在 **A 級以下**有競爭力，**A 級以上被 RWD 與 AWD 主宰**

### 1.4 寬體（Wide Body）

- 改善操控、可塞更寬的胎（**價值極高**）
- 副作用 1：**寬體有時阻力（drag）增加太多**
- 副作用 2：**可能擋住可調前下壓力**（極強的調校工具）
- **每輛車都要實驗**——不能一概而論

---

## 二、Aero（空力）

> 兩個重要分類：**adjustable（可調）** 與 **non-adjustable（不可調）**。

### 2.1 可調空力

- 可調空力允許在調校選單微調下壓力
- **可調前空力**（adjustable front aero）是車上**最強的升級之一**，特別在高級距
- **見到「降低 drag」的零件 → 一律拿**
- **見到「增加 drag」的零件 → 不要裝**

### 2.2 可調後翼

- 經常**只費極少 PI 甚至負 PI** → 多數 build 都很值得裝
- 但部分情況增加的 drag 不值得
- **高級距較常用**，**低級距可實驗看看**

---

## 三、Tires（輪胎）—— 改裝最關鍵的決策

### 3.1 胎質（Tire Compound）—— 2025 meta 修正

> ⚠️ **顛覆 2021 版的關鍵**：

- **Stock 與 Street 胎**：低級距可用
- **Sport 胎**：**幾乎永遠是壞選擇**（almost always bad）
- **公路賽 build（B ~ S1 級）**：應該選 **Rally 胎**或 **Drift 胎**！
- **Semi-slick**：常被 Rally / Drift / Full Slick 擠掉，但仍有用
- **Slick race 胎**：在更高級距 still 有用
- **FWD build**：低級距可實驗高抓力胎，甚至**試 Drag 胎**（部分 build 有效）
- **Rally / 越野 build**：幾乎都用 **Off-road race 胎**；B 級以下 PI 太貴時保留原廠

### 3.2 胎寬（Tire Width）—— 2025 meta 修正

> ⚠️ **顛覆 2021 版的另一關鍵**：

- **後胎寬**：幾乎永遠值得升
- **前胎寬**：**完全相反，通常應保留原廠**
  - 特別是裝了**可調前空力**的時候
  - **僅在最後手段、為求前抓**或有 PI 餘裕時升級
  - **FWD build** 有 PI 餘裕可考慮

### 3.3 輪圈與輪距（Wheels & Track Width）

- 輪圈尺寸／重量影響 PI——**Meta build 常選大重輪圈**讓 PI 預算可挪它處
- 但效果差異不大，作者**只用輪圈做最後的 PI 微調**
- **Track Width（輪距）**：可分前後獨立調整 over/understeer 平衡
  - **前後同時加寬**通常操控更好且 PI 成本低
  - 用此做**最終 PI 微調**

## 四、Drivetrain（傳動）

### 4.1 Clutch（離合器）

- 取決於車況與你的助手設定
- **舊／慢車**：升級效益大
- **現代運動車**：很少需要
- **Manual + Clutch（手排含離合）模式**：**完全不要升級**
- **Auto / Manual clutchless 模式**：可升級

### 4.2 Transmission（變速箱）

- 多數新車的原廠變速箱已夠用
- **Sport 升級**（特別在老車）通常**很便宜或甚至降 PI**——是好選擇，且解鎖 final drive 調校
- **Race 變速箱**：當原廠齒比不佳、或只有 4~5 速時值得升，解鎖**完整齒比調校**
- **超過 7 速幾乎永遠不值得**

### 4.3 Driveshaft（傳動軸）

- 微小、便宜的升級——用來**補最後幾 PI**

### 4.4 Differential（差速器）

- **永遠升級**——免費，且解鎖 diff 調校
- **絕對不要用 Sport diff**
- 依 build 類型升到對應的 race / rally / off-road / drift diff

---

## 五、Platform & Handling（底盤與操控）

### 5.1 Brakes（煞車）

- 通常**昂貴且效率不高**
- 只升一級**有時很便宜或免費**——這時可選
- **Race brakes** 留在有 PI 餘裕時——**不算最差但通常也不是最好**

### 5.2 Suspension（懸吊）

- **高級距 build**：幾乎都該升 **Race 懸吊**
- **越野 build**：升 **Rally 懸吊**
- **低級距車**：常常**保留原廠或用 Sport**，把 PI 留作他用
- **特殊情況**：公路車有時想用 **Rally 懸吊**取得更軟的彈簧率與更多懸吊行程
- ⚠️ **少數車原廠有四輪轉向（four-wheel steering）**，**升級懸吊會移除此功能** → 應保留原廠

### 5.3 Anti-Roll Bars（前後防傾桿）

- **每輛車都升到頂**
- 解鎖極有價值的調校設定，且**幾乎都是免費升級**

### 5.4 Chassis Reinforcement（車身強化）

- **從 meta 角度看少有價值**
- 可作為**配重（ballast）**讓你能塞下其他升級
- 多數情況**多出的重量不值得**
- 但對部分**老車**能讓駕駛感大幅變好——若想增車身剛性可裝

### 5.5 Weight Reduction（減重）

> 此處可能要做選擇。

- **PI 餘裕大** → 全力減重，價值極高
- **PI 已接近級距上限** → 在「加馬力」與「減重」之間找平衡
- 通則：**減重給更好的操控與加速**，**馬力給更高的長直線速度**

## 六、Engine（引擎）—— 把剩餘 PI 填進來

升級價值排序：

| 階段 | 升級 |
|---|---|
| **第一優先** | Exhaust（排氣）／Intake（進氣）／Displacement（排量） |
| **第二優先（補餘裕）** | Fuel system（燃料）／Valves（汽門）／Pistons（活塞）／Ignition（點火）／Flywheel（飛輪） |
| **保留作為 PI 餘裕大時或跳級用** | Turbo / Supercharger 升級 |
| **通常跳過**（除非大量餘裕） | Intake Manifold（進氣歧管）／Cams（凸輪）／Oil & Cooling（油冷）／Intercooler（中冷） |

## 七、最終提示

### 用小升級補最後幾 PI

> **Flywheel、Driveline（傳動軸）、輪圈選擇** 都是補或刮掉最後幾 PI 的好工具。

### 別低估「選對胎」的價值

> 跑完整流程後車仍覺得太慢 → **大概率是胎質要改**。

### 永遠實驗 power vs weight 的平衡

找到最佳**馬重比（power-to-weight ratio）**——這是 build 最大化的關鍵。

### 別怕問

社群中老玩家通常很樂意幫忙，只要你願意聽。

## 八、與 2021 版的關鍵差異整理

| 主題 | 2021 觀點 | 2025 修正 |
|---|---|---|
| 公路賽胎質 | Street → Sport → Semi-slick → Slick | **B~S1 公路改用 Rally/Drift 胎**；Sport 胎幾乎永遠是壞選 |
| 前胎寬 | Understeer 時必選；FWD 必升 | **預設保留原廠**，僅在沒裝可調前 aero 或 FWD 有餘裕時升 |
| 引擎置換 | 「能不換就不換」 | 仍以原廠優先，但**指明三組強引擎**：6.2 V8 / Racing V12 / 2L I4 VVT |
| 進氣置換 | NA 為先 | NA 為先，但**指明 Centrifugal Supercharger 是最強**置換 |
| Race brakes | 「幾乎都裝」 | 改為「PI 有餘裕再裝」、「升一級若便宜可升」 |
| 低級距懸吊 | 一律 race / rally | 「**低級距常保留原廠或 Sport**，把 PI 留作他用」 |
| 四輪轉向 | 未提及 | 新警告：**升懸吊會移除原廠四輪轉向** |
