## 一、級距系統（Class System）

每輛車旁邊的字母即為**級距（Class）**，數字則為**性能指數（Performance Index, PI）**。升級會抬高 PI，可能跨入更高級距。級距範圍依字母對應一段 PI 區間。

### 為什麼級距重要

- **單人模式**：幾乎所有單人內容都會依玩家當前 PI 動態縮放難度。想全程用 D 級老車跑完所有比賽完全可行，難度與用 X 級超跑一樣（也許慢一點）。**單人模式請以「好開、開得爽」為目標，不必追求極致 min-max 與 meta 調校。**
- **多人模式**：幾乎所有多人內容都依級距分組。S1 級車**永遠**不會與 S1 級以外的車對戰（無論是 Horizon Tour、開放賽、週賽都一樣）。

### 多人模式的關鍵原則

> **若要上線，PI 一定要拉到該級距的最高值**——任何低於該級距上限的 PI 都是浪費的性能。

重點不是把車本身升到最頂，而是**找到適合那輛車的級距，並在該級距內把 PI 拉滿**。

### 升級限度的經驗法則

- 大多數車**升 2 個級距以上**會開始變得難開
- 例：原廠 C 級的 Ford Escort 升到 S1，可能跟不上原本就在 A 級或 S1 的同級對手
- 有例外但不多

## 二、Conversions（置換類零件）

很多 build 不會動到此區，但幾乎每輛車至少有引擎置換或驅動置換可選，部分車還有寬體與進氣置換（Turbo/Supercharger）。

### 引擎置換（Engine Swap）

- **預設原則：能不換就不換**——盡量保留原廠引擎
- **何時該換**：原廠引擎馬力不足以推進到目標級距時
- **特殊技巧**：少數情況下換引擎反而**降 PI**，若想把車降級可留意

### 驅動置換（Drivetrain Swap）

- 後驅車多半可換 AWD
- AWD 車多半可換 RWD
- 前驅車通常 AWD、RWD 兩者都能換

**預設仍是原廠最佳**。
- 過去 Horizon 系列 AWD 置換非常強勢，FH5 仍有一定強度
- 但 FH5 中 RWD 與 FWD 的線上競爭力已經很高，多半不需置換
- 結論：**通常不換**，依個人選擇

### 進氣置換（Aspiration Swap）

部分引擎可換 Turbo 或 Supercharger。
- **若不加增壓就能達到目標馬力，優先選擇 NA**——FH5 多數情況 NA 馬力比較好用
- 馬力不夠或還有 PI 餘裕時，再加增壓
- **Turbo**：給更高峰值馬力，但只在高轉速段提升
- **Supercharger**：峰值馬力較低，但**整段轉速範圍**都提升

### 寬體（Wide Body Kit）

- 改善外觀之外，也透過增加**輪距（track width）**改善操控
- 注意：**裝寬體後通常無法再加可調前擾流（adjustable front aero）**
- 作者實機示範時先不裝寬體以展示前擾流效果，最終 build 才裝回

### Conversions 結論

絕大多數 build 不需動 Conversions。
> **新手通則：想要 V8 就買原廠 V8 的車；想要 AWD 就買原廠 AWD 的車。** 因為車的重量分配與懸吊數值都圍繞原廠引擎與驅動配置設定。並非死規矩，但是新手起手最穩的方式。

## 三、Aero（空力套件）

### 可調前擾流（Adjustable Front Splitter）

- 幾乎每輛車都有「Forza 招牌」的前擾流＋後翼可選
- **前擾流通常很貴**，但在 **A 級以上**的高級距車**非常重要**
- **能不裝就不裝**，但若 build 嚴重 understeer 且前抓不足，就裝
- 注意：寬體有時會擋住可調前擾流，需取捨

### 可調後翼（Adjustable Rear Aero）

- 部分情況下**反而降 PI**——這時就裝吧，調校時把多餘下壓力降回去就好
- 多數情況會升 PI——這時只在後抓力嚴重不足時才裝
- 高級距車（A、S1、S2）下壓力的價值很高

### 不可調車身件（Bumpers / Side Skirts）

- 用來在 build 末端微調 PI 的好工具
- 部分件可加減 1~2 PI 而不影響實際性能
- 若車剛好**超過級距上限 1~2 PI**，加裝稍重的車身件就能拉回級距內

## 四、Tires & Rims（輪胎與輪圈）

> 此區塊是改裝中**第一個重要的決策區**。

### 胎質（Tire Compound）

| 用途 | 建議胎質 |
|---|---|
| 拉力／越野／漂移 build | 對應的 **rally / off-road / drift** 胎，效益巨大，除了極少數低級距 rally build 可保留原廠胎 |
| 越野（純越野 build） | **Off-road 胎**——不只卡車與越野賽用，做混合路面 rally 也好用，前提是不會大量壓路 |
| 公路賽（Road Racing） | Street → Sport → Semi-slick → Slick race（依車況可選範圍會不同；原廠胎可能已優於 street/sport） |

### 公路胎升級的經驗法則

- FH5 的胎升級成本比 FH4 便宜，但效益也沒那麼絕對
- **AWD build**：通常原廠胎就夠用，除非馬力升太多
- **RWD / FWD build**：至少升一級胎質通常很值得
- **Slick race 胎**：**幾乎不適合線上對戰**，比較適合 Rivals 賽
- **Semi-slick 胎**：比 Slick 便宜，雨天或臨時下越野時表現更好

### 胎寬（Tire Width）

- **前胎寬**：通常成本不低但若 understeer 就值得；FWD build 必選
- **後胎寬（前驅／四驅）**：升級通常不太升 PI，甚至因為加重而降 PI
- **後胎寬（後驅）**：能加大量抓地，但 PI 成本較高
- **總原則：後胎寬升到頂幾乎永遠值得**

### 輪圈樣式（Rim Style）

- 不同輪圈重量不同 → 可微調 PI（與不可調車身件同理）
- 偏重的輪圈若能讓你多塞下一個馬力升級就值得

### 輪圈尺寸（Rim Size）與輪距（Wheel Spacing）

- **大尺寸輪圈**讓車更「跟手」（responsive），通常幾乎不費 PI
- **Wheel Spacing（輪距加寬）**便宜且明顯改善操控與抓地
- **後驅 build 技巧**：前留原廠 spacing，**只加大後 spacing** → 後抓更高、車更穩
- 用 spacing 調整轉向／轉向不足平衡幾乎不費 PI：
  - 後 spacing 大於前 → 偏 understeer，更穩
  - 前 spacing 大於後 → 偏 oversteer，更靈動
  - **警告**：前輪距比後輪距大太多會破壞懸吊運作，慎用

## 五、Drivetrain（傳動）

### Clutch（離合器）

- **Manual w/ Clutch（手排含離合）**：升級價值降低
- **Manual Clutchless（手排無離合）／ Auto（自排）**：升級非常有用，尤其是老車慢車
- 升級時關注換檔時間（shift time）的變化

### Transmission（變速箱）

- **Sport Transmission**：幾乎都該升——便宜且解鎖**最終傳動比（Final Drive）**調校，**Final Drive 是變速箱調校中最重要的設定**
- **Race Transmission**：只在需要更多檔或有 PI 餘裕時升級，可調個別檔位齒比

### Driveline（傳動軸）

- 微調 PI 的好工具——掉重量、加快引擎反應、通常幾乎不費 PI

### Differential（差速器）

- **免費升級！** 升級的 diff 不費 PI 並解鎖 diff 調校
- **Sport diff**：在多數 build 上**沒用**，直接跳到 race 或 rally
- **Rally / Off-road build**：用 rally 或 off-road diff
- **Off-road diff**：只用在純越野 build（沙丘越野車、攀岩車）

## 六、Platform & Handling（底盤與操控）

### Brakes（煞車）

- 通常很值得且差異明顯
- **作者幾乎所有 build 都裝 race brakes**——貴但值得

### Suspension（懸吊）

- 同樣 race / rally 等級**幾乎一律值得**
- 不只大幅改善操控，更**解鎖大量調校設定**

### Anti-Roll Bars / ARBs（防傾桿）

- 通常**免費或只費 1~2 PI**
- 解鎖 **ARB 調校**——這是**非常重要的調校設定**
- **沒理由不裝**

### Roll Cage（防滾架）& Chassis Reinforcement（車身強化）

- **避免裝完整防滾架**——即使它有時降 PI，重量增幅大且操控提升極小
- 例外：把非常老的低級距車塞進高級距（加大量馬力）時值得
- **車身強化（部分等級）規則**：
  - 升 1 個級距 → 街道級強化（street）
  - 升 2 個級距 → 運動級強化（sport）
  - 從 B 級拉到 S2 → 完整防滾架

### 減重（Weight Reduction）

- 可能是最貴的升級之一，原因充分
- **取捨關鍵**：到底是「加馬力」還是「減重量」
- **Forza 偏好 min-max**——只減一點＋只加一點通常不獎勵
- **PI 接近上限時**：要嘛**全力減重**，要嘛**完全跳過減重，全部加馬力**
- 操控上：減重最大化更好開
- 線上競爭：**馬力派 build 通常更受歡迎**
- 經典格言：「**加馬力讓你直線更快；減重讓你哪裡都更快**」

## 七、Engine（引擎）

> 這是最後最有趣的部分。其他區處理完，剩下的 PI 全砸這裡。

### 升級順序原則

1. **先解鎖調校項目**（懸吊、煞車、ARB、diff 等等）
2. **把胎與懸吊配置調好**
3. **剩下的 PI 全進引擎**

### 引擎內部升級價值排序（作者偏好）

| 升級 | 價值 | 備註 |
|---|---|---|
| **Intake（進氣）/ Exhaust（排氣）** | 極高 | 高 CP 值 |
| **Oil & Cooling（油冷）** | 高 | 通常便宜，因為加重 |
| **Intercooler（中冷器）** | 高（有 turbo/supercharger 時） | 便宜，因為加重 |
| **Supercharger 升級** | 中（若是機械增壓車） | 加馬力直接 |
| **Turbo 升級** | 留最後 | 馬力提升大但較貴 |
| **Displacement（排量）** | 留最後 | 同樣是大馬力來源 |
| **Cams（凸輪）** | 不太值得 | 改高紅線，但通常太貴 |
| **Fuel / Ignition / Compression（燃料／點火／壓縮）** | 補餘力 | 有 PI 餘裕再加 |
| **Flywheel（飛輪）** | 末段微調 | 改善反應，幾乎不費 PI |

### 達到級距上限後的小技巧

回頭檢查那些**不會升 PI 的升級**——輪圈、胎寬、傳動軸 reinforcement 等，看看是否能再多加一點而不超 PI。例如作者示範車的 drive shaft 升級**完全不費 PI**。

---

## 八、總結流程

1. **先解鎖調校選項**並裝上正確的**胎、懸吊、變速箱**
2. PI 還有大量餘裕時：**裝煞車、離合器，必要時加可調 aero**
3. **剩餘 PI** 用以下二擇一填滿：
   - **減重派**：輕輪圈、傳動軸、Weight Reduction 升級
   - **馬力派**：堆 Engine 升級

希望這份指南能幫你在 FH5 線上對戰中明白該優先升級什麼、為什麼。

## 速查表（額外整理）

### 級距內 PI 拉滿的微調工具

| 工具 | 效果 | PI 影響 |
|---|---|---|
| 輪圈樣式 | 增重/減重 | ±少量 |
| 不可調車身件 | 增重/減重 | ±1~2 |
| 傳動軸（Driveline） | 減重、加快引擎反應 | 幾乎為 0 或負 |
| Flywheel | 改善引擎反應 | 0~少量 |
| 後翼（adjustable rear） | 部分情況**負 PI** | -X 或 +X |

### 防滾架／車身強化決策表

| 升幾個級距 | 建議裝什麼 |
|---|---|
| 0~1 級 | 不裝 |
| 1 級 | Street 強化 |
| 2 級 | Sport 強化 |
| 3+ 級（如 B → S2） | Full Cage |
