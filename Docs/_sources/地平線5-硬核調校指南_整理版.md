## 一、基礎概念

### 1.1 什麼是調校

在地平線 5 裡，「調校」指的是透過**改裝、升級配件與對應的參數設定**，讓車輛在性能參數（PI）區間內綜合性能最大化，以適應不同類型的賽車與賽道。

### 1.2 PI 等級

- 地平線 5 的性能參數分為 **D、C、B、A、S1、S2、X 共 7 個等級**。
- 對於**公路、拉力、越野**賽事，通常要讓車輛在目標等級內的 PI **最大化**：
  - **A 級** → 升級到 800
  - **S1 級** → 升級到 900
  - 其他等級以此類推（B=700、S2=998 等，依遊戲上限）。

### 1.3 轉向過度 vs. 轉向不足

| 現象 | 說明 | 俗稱 |
|------|------|------|
| **轉向不足**（Understeer） | 轉彎幅度**小於**前輪轉角，車頭向外側滑 | 推頭 |
| **轉向過度**（Oversteer） | 轉彎幅度**過大**，車尾向外側滑 | 甩尾 |

調校的目的，就是在**速度與操控之間取捨**，在**轉向過度與轉向不足之間找平衡**，最後調出適合自己的手感。

## 二、改裝配件選擇（PI 性價比思路）

重點：**提高性能的同時消耗的 PI 最少**。

### 2.1 發動機

PI 性價比最高的發動機（優先考慮）：

- **6.2L V12 Racing**、**V12 Racing**
- **V8（5.2L 或 8.4L）**
- **V10**
- **3.2L 直列六缸**
- **1.6L 與 2.0L 直列四缸 VVT**（可變氣門正時）

每輛車可置換的引擎列表不同，看到上述型號優先考慮，再依目標 PI 等級挑選合適的。

### 2.2 進氣增壓

- **離心增壓** > 渦輪增壓、機械增壓（PI 性價比最高）
- 有離心增壓選項就優先用。

### 2.3 傳動系統

- 通常**保持原廠**，或升級為**四驅**。

### 2.4 車身 / 空氣動力套件

- **寬體套件**：全方位改變，要看消耗多少 PI、增加多少重量，具體情況具體分析。
- **前後可調空力套件**：增加下壓力 → 提升操控與彎道性能，但降低極速。
  - **前驅、四驅**：可調尾翼能提升彎道性能，還能降 PI。
  - **後驅**：可調尾翼通常也必須，除非後輪抓地力已足夠，或是車重極速的大馬力後驅調校（為保留直線優勢才不裝尾翼）。
- **可調前保險桿**：PI 性價比較低，但能顯著改善操控；嚴重推頭的車可加裝。
- **側裙、後包圍**：通常用來最後**配重**與**微調卡 PI 上限**。

### 2.5 輪胎

- **胎種**（依車種）：
  - **公路調校** → 使用**拉力胎**（不同路面與天氣對其性能影響最小）。
  - **拉力、越野調校** → 直接用**越野胎**。
- **胎寬**：越寬 → 接觸面積越大、抓地力越強。
  - **四驅、後驅**：只升級**後胎寬**，前胎寬性價比極低（除非最後還有 PI 剩餘）。
  - **前驅**：反之，**優先升級前胎寬**。

### 2.6 輪圈與輪距

- **輪圈款式與大小**：用來配重。
  - **後輪圈加大**通常能有效降低 PI。
  - **前輪圈**效果因車而異，不少車前輪圈加大會**同時增重並增加 PI**，要看。
- **輪距**：
  - **後輪距** → 升級到最寬。
  - **前輪距** → PI 性價比極低，不優先考慮。

### 2.7 傳動 / 變速箱 / 差速器

- **離合器**：升級能改善換檔速度，但 PI 性價比極低。若使用**手動離合**，則**不要升級離合器**。
- **變速箱**：是**增重與卡 PI 的重要手段**。
  - **跑車變速箱**：解鎖「最終傳動比」調校。
  - **賽車變速箱**：解鎖**所有檔位**的調校。
  - 要改賽車變速箱 → 直接套用**十速變速箱**；若有 PI 空間再選擇檔位較少的。
  - 部分**老爺車 / 復古車**變速箱升級消耗 PI 極大 → **沿用原廠**。
  - 有些車原廠齒輪比就很完美，原廠即可；有些車升級街車或跑車變速箱反而**能減 PI**，要仔細比對。
- **傳動軸升級**：主要用來**增重**，需要配重時再考慮。
- **雙向可調差速器**：**必裝**。不同類型的雙向差速器僅初始參數不同。

### 2.8 底盤與操控

- **剎車**：PI 性價比極低。
  - 越野、拉力 → **完全不用**升級剎車。
  - 漂移、直線調校 → 升級剎車主要用於減速與漂移控制。
  - 公路調校 → 通常**不需要**升級剎車；有些車自帶最高級可調剎車。
  - 「車剎不住」通常**不是車的問題**，是車手的問題：錯過剎車點、輪胎 ABS 或已失去抓地力——此時再剎反而雪上加霜。剎車唯有在輪胎具備機械抓地力時才能有效減速。

- **彈簧 / 懸掛**：
  - 跑車板懸掛：通常只消耗 1 格 PI，但只是簡單降低車身高度、不可調，與原廠無異，**性價比很低**。且地平線 5 裡車身高度對公路操控影響其實不大。
  - **拉力、越野、直線加速調校** → 直接用**拉力懸掛**。
  - **漂移調校** → 用**漂移懸掛**。
  - **公路調校**花樣較多，具體分析：
    - **A 級以下**：一般用**原廠懸掛**。
    - **A 級、S1 級**：若需升級，優先選 PI 低的；**賽車懸掛**與**漂移懸掛**本質相同，只是預設值不同，拉力懸掛較軟、重心轉移較快。
    - **S2 級**：常用原廠或賽車懸掛（很多 S2 車原廠就已經是可調賽車懸掛）。

- **前後防傾桿**：**必裝**。防傾桿的調校是改善操控最直接的手段之一，其中**前防傾桿尤其重要**。

- **底盤強化**：因車而異。
  - 老車：結構原因，底盤強化帶來的全方位提升較顯著，但消耗 PI 也較多。
  - 現代一體成型底盤 / 碳纖維一體化車身：底盤強化的提升遠不如**減重**的收益。

- **減重**：能全面提升性能，依目標 PI 與車重決定減多少。
  - **輪胎抓地力足夠時**，大馬力調校可**不減重**。
  - 偏操控的調校 → **優先最大減重**。
  - **不要害怕走極端**：減一點重量、加一點馬力都帶不來質變；改車前先想好思路，**要麼 all in 馬力，要麼梭哈減重加操控**。
  - 拉力、漂移、直線加速調校 → 優先**最大幅度減重**。

### 2.9 發動機各子系統

性價比排序（由高至低）：

1. **排氣系統**、**節氣門**（最高性價比）
2. **增壓系統**：
   - 離心增壓 → 優先升級**機械增壓**（升級性價比較高）
   - 渦輪增壓 → **最後再考慮**
3. **燃料**、**點火**、**氣缸**（不減速的 PI）
4. **氣閥**、**活塞**
5. **進氣過濾**、**凸輪軸**：能提高轉速 → 拉寬功率帶、提升極速，但消耗 PI 很多，**通常不優先**
6. **中冷器 / 油冷器**：會**大幅增加重量**；但有些車中冷 / 油冷升級能**反而減 PI**，要精算增重、減重與 PI 收益。
7. **飛輪**：改善油門響應，對前中段加速有幫助，但性價比不高，通常只用來配重與卡 PI。

### 2.10 改裝順序建議

1. 從改裝選單開始，先選好**輪胎、傳動、懸掛、空力**與必須升級的配件。
2. 最後再**對齊馬力**或**極致減重**。
3. 初次改裝後目標：把 PI 壓在**上限 −4**的範圍內。
4. 組合搭配以下**配重 / 卡 PI 配件**榨乾剩餘 PI 空間：
   - 不同重量的輪圈
   - 輪圈大小
   - 傳動軸
   - 變速箱
   - 中冷器 / 油冷器
   - 飛輪

## 三、調校參數設定

> 地平線 5 的調校就是很極端的——**不要害怕走極端**。

### 3.1 胎壓

- **低胎壓** → 增加與地面的接觸面積、增加抓地力，但過低會因過熱變形而適得其反。
- **高胎壓** → 減少接觸面積、增加轉向靈敏度、降低升溫速度，但過高會失去抓地力。
- **轉向靈敏度 ≠ 轉向過度／不足**：轉向靈敏度指的是輪胎轉彎的靈活度。
- **胎壓前低後高** → 偏轉向過度。
- **胎壓前高後低** → 偏轉向不足。

#### 建議初始胎壓

| 調校類型 | 車重 | 前後胎壓 |
|---|---|---|
| 公路賽 | < 1000 kg | 前後 **1.5** 磅 |
| 公路賽 | 1000–1400 kg | 前後 **1.8** 磅 |
| 公路賽 | > 1400 kg | 前後 **2.0** 磅 |
| 拉力 / 越野 | — | 在公路基礎上 **−0.3** |
| 直線加速 | — | **前最高、後最低** |
| 漂移 | — | 通常**前高後低**，具體數值因車與習慣差異很大 |

比賽中輪胎會升溫達到理想胎壓。

### 3.2 變速箱齒輪比

#### 原理：功率帶（Power Band）

- 改裝頁面右側的 **Engine Dyno Graph**：
  - **紅線** = 扭矩（決定加速度）
  - **黃線** = 馬力 / 功率（決定速度）
  - **橫軸** = 發動機轉速
  - **縱軸** = 發動機輸出功率
- **峰值扭矩 → 峰值馬力**之間的轉速區間，就是該發動機的**理想工作轉速（功率帶）**。
- 例：圖中引擎理想轉速大約 **5000–9300 rpm**。

#### 齒輪比調校原則

- 每一根斜線 = 每一檔的走勢；起點為該檔起始轉速、終點為紅線上限。
- **橫軸**：車速；中點標注的是該車目前 PI 下的理論最高車速。
- **縱軸**：發動機轉速。
- 目標：**從二檔開始，每一檔最高轉速都落在理想功率帶內；每檔起始轉速依序遞增。**
- 調整**最終傳動比**，讓車輛在**最後一檔**拉到功率 / 扭矩峰值時剛好達到極速——也就是最後一檔末端**頂到右上角**。
- 理論極速只有在**無額外下壓力**、拉力與理想最終傳動比下才能達到。

#### 常見情況

- 選了**十速變速箱**只是為了配重／卡 PI，**不代表要用十個檔**——賽車中除非萬轉引擎，否則很難用到 8 檔以上。此時透過最終傳動比，把十速當**六檔或七檔**用。
- **起步、出彎加速打滑**：可**把低檔拉長**
  - 賽車變速箱 → 把**第一檔**的傳動比調長。
  - 跑車變速箱 → 把**最終傳動比**調長。

> UP 主影片中另分享了幾個「萬能傳動比」，可直接套用，再依上述邏輯調整最終傳動比。

### 3.3 輪胎定位

#### 外傾角（Camber）

- 輪胎縱軸與地面的夾角：垂直=0、頂端內收=**負外傾**、頂端外張=**正外傾**。
- 調外傾角的目的：讓**彎中外側輪胎**與地面的接觸面積最大化 → 提供最大抓地力。
- **並不是負值越小越好**，需根據**遙測（Telemetry）**反饋的數據調整。

#### 束角（Toe）

- 輪胎縱向與車頭指向的夾角：正前=0、前端外張（外八）=**正束角**、前端內收=**負束角**。
- 每 **0.1** 的差距都會顯著影響加速、減速與胎溫。
- **正束角** → 輕微改善轉向不足，過大會不穩定。
- **負束角** → 讓車保持直線。

建議：

| 驅動方式 | 前輪束角 | 後輪束角 |
|---|---|---|
| 前驅 / 四驅（天生推頭） | 加不超過 **+0.5** 外束角 | 加不超過 **+0.2** |
| 大馬力後驅 | — | 加一點**負束角**（提升後輪穩定性） |

#### 後傾角（Caster）

- **初始直接拉到最大值 7**，若轉向太靈活最後再微調。
- 後傾角越小 → 前輪需要更多負外傾。

#### 各類型調校的建議值（前外傾 / 後外傾 / 前束 / 後束 / 後傾角）

| 調校類型 | 前外傾 | 後外傾 | 前束 | 後束 | 後傾角 |
|---|---|---|---|---|---|
| **公路** | −0.5 | −0.5 | 0.3 | 0.1 | 7 |
| **拉力 / 越野** | 0 | −0.5 | 0 | 0 | 7 |
| **漂移** | −（負外傾） | −（負外傾） | 2 | 1 | 7 |
| **直線加速**（含翹頭、翹屁股衍生） | 0 | 0 | 0 | 0 | 7 |

### 3.4 防傾桿（Anti-roll Bar）

> 改善**入彎**與**彎中操控**最直接的工具，**必須升級可調防傾桿**。

原則：

- **前防傾桿越軟、後防傾桿越硬** → 改善轉向不足。
- 過彎中出現甩尾 → **調高前防傾桿**。

| 調校類型 | 前防傾桿 | 後防傾桿 |
|---|---|---|
| 前驅 / 四驅（基本值） | **1** | **65** |
| 四驅（改善彎中穩定性） | **10** | **6.5** |
| 前驅（改善彎中穩定性） | **1** | **5.6** |
| 後驅（初始） | **10** | **5.6** |
| 拉力 / 越野（初始） | **10** | **2.5** |
| 大皮卡（如 Mercedes X-Class） | **1** | **最軟** |
| 直線加速 | **65** | **1** |

### 3.5 彈簧

> 與防傾桿一樣，是**改善操控最直接的工具之一**。

原則：

- **前軟後硬** → 改善轉向不足。
- **彈簧越軟** → 重心轉移越快。
- **彈簧越硬** → 操控響應越好，但太硬容易失去抓地力。
- **前硬後軟** → 減少轉向過度、增加轉向不足。

| 驅動方式 | 前彈簧 | 後彈簧 |
|---|---|---|
| **前驅 / 四驅** | 最軟 | 約在 **2/3** 位置 |
| **四驅（另一思路）** | 最軟 | 最硬 |
| **後驅** | 總體**不超過調整範圍的 1/3**，初始約 80 | 初始約 100（依試車調整） |
| **大馬力後驅** | 前後硬度可**一樣**，甚至都用最軟 | 同左 |
| **拉力懸掛（共通）** | 前彈簧不超過行程的 **1/6** | 後彈簧不超過 **1/4** |
| **拉力調校** | 偏軟 | 偏硬（前軟後硬提升轉向） |
| **越野調校** | 前後軟硬度**一樣**（起跳時不推頭、不翹頭） | 同左 |
| **輕量越野車** | 前後都最軟 | 同左 |
| **直線加速** | 最硬 | 最軟 |

> ⚠️ 遊戲早期流傳的「四代彈簧計算公式」計算出來的結果有**嚴重轉向不足**，**不推薦使用**。

### 3.6 車身高度

| 調校類型 | 設定 |
|---|---|
| **公路**（一般） | 用**預設數值**，以「高的一側」配平即可 |
| **公路特例**（KSG CCGT、法拉利 FXX 等） | 可**拉高車頭**，提高操控可控性，甚至提升極速 |
| **拉力 / 越野** | 前後都**最高** |
| **直線加速** | 前輪**最高**、後輪**最低** |
| **漂移** | 前後**統一最低** |

### 3.7 阻尼（Damping）

- **回彈硬度（Rebound）**：彈簧壓縮後復原的速度。越軟 → 回彈越快。
- **壓縮硬度（Bump）**：彈簧被壓縮的速度。越軟 → 被壓縮得越快。
- **回彈速度永遠大於壓縮速度**。
- 理論上 **壓縮硬度 = 回彈硬度的 50–75%**；解鎖高級懸掛後預設通常落在 **63%**，但遊戲實測太硬，不可取。
- 地平線 5 較青睞**偏軟**的回彈與壓縮——**壓縮越軟，吸震能力越強**。

#### 建議範圍

| 項目 | 建議 |
|---|---|
| **回彈** 一般範圍 | **8 ~ 14** |
| **壓縮（拉力 / 越野）** | 回彈 × **0.25** |
| **壓縮（公路）** | 回彈 × **0.4** |

初始設定：可先用預設回彈值，再乘以 0.25 / 0.4 作為壓縮值。

#### 調整邏輯

| 症狀 | 調整 |
|---|---|
| 剎車或落地後常被彈飛失控 | **回彈調硬** |
| 過路肩時輪胎彈起 | **壓縮硬度調軟** |
| 改善轉向不足 | 回彈**前軟後硬** |
| 改善轉向過度 | 回彈**前硬後軟** |
| 前驅 / 四驅的一般作法 | 在上述範圍內**前軟後硬** |
| **直線加速** | **前 20 / 後 1** |

### 3.8 空氣動力（下壓力）

> 右側說明已經很清楚，原理不難。

| 驅動方式 | 前下壓力 | 後下壓力 |
|---|---|---|
| **前驅 / 四驅** | **拉到最高** | 不超過**一半**位置 |
| **後驅**（A 級以下） | **拉滿** | 可先視為**最低**，再依試車逐步調大 |
| **後驅**（S1 以上大馬力） | 拉滿 | 依需求偏大，甚至**拉滿** |
| **直線加速** | — | **不需要**升級空力套件 |

原則：

- 後下壓力越大 → 後輪抓地力越強，但太大會造成**轉向不足**。
- PI 等級越高的車 → 後下壓力需求越大。
- 初始設定都需要在**試車過程中依反饋調整**。

### 3.9 剎車

| 項目 | 設定 |
|---|---|
| **剎車平衡**（一般） | **> 50%**（偏前 → 增加剎車時的轉向過度；偏後 → 減少轉向過度、增加轉向不足） |
| **剎車平衡**（特例，如第六元素特別版、CCGT 這類天生轉向靈活的車） | 可設 **< 50%** 以提升剎車穩定性 |
| **制動力（手柄 / 方向盤，無 ABS）後驅** | **60 ~ 80%**（降低行程以利循跡剎車、精細控制） |
| **制動力（鍵盤後驅）** | **60 ~ 80%** |
| **制動力（開 ABS）** | **80 ~ 200%** 都可，比較靈活 |

- **制動力低** → 增加剎車動作行程，對循跡剎車與精細控制有利。
- **制動力高** → 增加剎車力度、減少行程。

### 3.10 差速器（重點，影響極大）

#### 基本原理

- 差速器控制**左右輪的扭矩分配**。
- 裝了雙向差速器後：
  - **四驅** 有三個板塊：**前差速器**、**後差速器**、**中央差速器**。
  - **前驅** 只有前差速器；**後驅** 只有後差速器。
- **0%** = 差速器完全放開，兩邊輪胎各轉各的。
- **100%** = 差速器完全鎖死，左右兩輪轉速始終相同。
- **加速（Accel）設定** → 控制**加速時**差速器鎖死的快慢。
- **減速（Decel）設定** → 控制**鬆油時**差速器鎖死的快慢。
- **中央差速器** → 控制扭矩的**前後分配**。
  - **100%** = 扭矩全部輸出到後軸 → 越接近 100% 越偏後驅。
  - 越接近 0% 越偏前驅。

#### 實驗驗證（按 Telemetry 觀察輪胎抓地力畫面）

- 綠圈 = 輪胎抓地力（懸空輪胎沒有綠圈）。
- 地平線 5 裡：
  - **差速器 0%** 時 → 動力全部分配到**抓地力較小**的輪子（懸空輪猛轉、接地輪沒動力）。
  - **前差加速 50%** → 接地的前輪開始轉動，轉速理論上達到另一輪的一半。
  - **100% 鎖死** → 左右前輪轉速相同，可輕鬆開出困境。
- 結論：**四驅配置下中央差速器拉到最左或最右，都不會完全變成純前驅或純後驅**。

#### 調校原則

現實過彎中，**內輪轉速需小於外輪**才能靈活轉向（這正是人類發明 Torque Vectoring／扭矩矢量系統的原因）。

- **完全鎖死不可取**，合理的鎖定速度才是目標。
- 正因如此，**後驅漂移** 常用 **後差加速 100% / 後差減速 90%** 的組合——鎖得越死，車越容易飄。

#### 後差速器

- **後差加速** 數值越高 → 加速時鎖定越快 → **減少轉向過度**，但**增加轉向不足**。
  - 馬力越大 → 需要的後差加速越大，一般在 **50 ~ 95%**。
- **後差減速** → 控制鬆油時鎖定速度 → 有效調整**入彎穩定性**。
  - 數值越大 → 減速入彎越穩定。
  - 拉力 / 越野若在減速轉彎時常甩尾掉頭 → **調高後差減速**。

#### 前差速器

- **前差加速設定需要大量試車才能確定**。
  - **偏低** → 彎中與出彎操控更好；但低速彎入彎會推頭，中高速彎中與出彎會甩尾。
  - **偏高** → 改善低速彎入彎的推頭；但增加彎中與中高速彎出彎的推頭。
- 公路調校一般在 **20 ~ 90%**。
- 大馬力 S1 / S2 車型偏高的前差加速能讓彎中、出彎更穩。
- 拉力一般 **30 ~ 90%**；越野 **80 ~ 100%**。
- **前差減速** → 一般設為 **0**；若已調高後差減速仍推頭，可稍微調高。
- 通常 **前差減速不高於前差加速**。

#### 中央差速器

| 調校類型 | 中差 |
|---|---|
| 公路（四驅） | **75%** 左右 |
| 拉力 | **60%** 左右 |

#### 差速器初始值

| 驅動方式 | 加速 | 減速 |
|---|---|---|
| **前驅** | 50% | 0% |
| **後驅** | 80% | 5% |
| **大馬力後驅**（需穩定性） | 調**低後差加速** | 適當調**高後差減速** |

## 四、Telemetry（遙測）使用指南

調校完成後要在試錯中驗證並優化，**遙測是最直觀的數據來源**。

### 4.1 開啟方式

- **預設鍵位**：進階控制選單可將「HANNA」替換為「遙測鍵」，之後按方向鍵下即可打開。
- **自訂鍵位**：只能用鍵盤開啟。

### 4.2 常用頁面

#### ① 一般（General）頁面

- **左側**：即時顯示發動機輸出功率、扭矩、轉速。
- **右側**：油門與剎車行程。
- **用途**：查看轉速、剎車抱死臨界值 → **優化齒輪比、練習剎車習慣**。

#### ② 懸掛系統（Suspension）頁面

- 紫粉色 = 懸掛壓縮程度，靜止為 50%。
- 地面不顛簸時，紫粉色若把白色「填滿」→ 表示懸掛已壓到底。
  - 解法：**調硬彈簧、調高壓縮硬度、或調高車身高度**。
- **理想行駛壓縮範圍**：懸掛行程的 **15 ~ 85%**。
- **平路上紫粉色波動幅度很大** → **彈簧調硬**。
- **波動幅度偏小** → **彈簧調軟**。

#### ③ 輪胎資料頁面（用得最多）

- 用來優化**輪胎定位**。
- 過彎時重點觀察**外側輪胎的外傾角**：
  - 理論上 **0°** 最理想。
  - 但路面不平，過彎時出現 **約 2° 的外傾角**都算合理（輪胎會形變，2° 時實際仍緊貼地面）。
  - **不用糾結**，只要不常出現**正外傾**就行。

#### ④ 輪胎溫度（Tire Temperature）頁面

- 即時反饋**內側 / 中間 / 外側**胎溫；輪胎形狀白框顯示各區溫度分佈。

| 顏色 | 狀態 |
|---|---|
| 淺藍 | 胎溫**太冷** |
| 透明 | 胎溫**尚可** |
| 半透明 / 黃 / 淺黃 | **理想工作溫度** |
| 橙紅 | 輪胎**過熱** |

- 用來驗證前面**輪胎定位 + 胎壓**的設定。
- 環境溫度、天氣、路況也會影響胎溫 → 建議在**勁敵模式（Rivals）** 中測試。
- **理想狀態**：
  - 比賽中輪胎**內側溫度 ≥ 中間 ≥ 外側**；
  - 長時間維持在**半透明 / 黃 / 淺黃**；
  - 剎車升溫後出彎能迅速降回淺黃；
  - **內外溫差 ≤ 10°C**。
- **溫差太大** → 負外傾角度過大 → **調小負外傾**即可。
