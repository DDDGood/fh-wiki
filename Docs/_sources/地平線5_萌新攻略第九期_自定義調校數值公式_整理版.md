## 〇、開場補充：飛輪（Flywheel）為什麼通常不用來加分？

承接上一期「自定義升級」的問題：為什麼 PI 分不算在飛輪上？

- 飛輪可以加 1～4 分（視車而定），但它的 **PI 性價比與變速箱相同：1 分 = 2 kg**，有時甚至變速箱還更划算。
- 所以原則上 **盡可能不裝飛輪**，優先用離合器把重量壓到最輕。
- **什麼時候才裝飛輪？** 等你把所有「加成類」配件都改完、重量已經減到最輕，但 PI 還差 1~2 分才到頂（例如 800 分上限還差一點）時，再把飛輪拿來湊數。

### 對照示例

- 裝了一個 +1 分的飛輪後，再回去把離合器拆掉變回 800 分
- 實際比對數據：
  - 雖然同樣是 1 分 = 2 kg 的輸出
  - 但 **離合器減掉的公斤數比飛輪多一點點**
  - 結果：裝飛輪的車，**會比直接裝離合器的車稍微重一點**（差距很小，但本質上離合器更輕）
- 所以若能維持「不裝飛輪 + 裝離合器」的組合，車體會是最輕的狀態。

## 二、拉力公式逐項詳解（為什麼這樣改？）

作者拿拉力車的公式作為範本，逐項解釋原理。以下是完整說明。

### 2.1 輪胎（胎壓）

**核心邏輯**：輪胎需要達到一定溫度才有抓地力；**胎壓越低 → 輪胎越早暖 → 越早有抓地力**。

- **低胎壓優點**：例如 1.0 胎壓，可能在速度 30~40 時輪胎就暖好、開始有抓地力。
- **低胎壓缺點**：同樣區間大小（例如 100 的溫度窗口），過早進入抓地區，到 130 就容易過熱；**輪胎過熱後發力下降**。
- **驗證工具**：鍵盤按 **T** 可打開遙測，查看輪胎溫度。

### 各車種選擇邏輯

| 車種 | 典型胎壓 | 原因 |
|------|---------|------|
| 越野 / 拉力 | **1.0 / 1.0** | 低速彎、甚至高速彎都希望能飄著過；飛坡後要快速有抓地力；土路飄著走甚至更快 |
| 公路 | **前 1.3 / 後 1.4** | 中高速彎（140、170~180 km/h）多，需要輪胎在該有抓地力時不要過熱；晚一點進入抓地區可延緩過熱 |

### 2.2 齒比（傳動比）——拉到最高再往回拉

齒比有兩部分：**各檔比值** 和 **終減速比（Final Drive，傳動比）**。

#### 各檔比值（固定公式）

| 檔位數 | 公式 |
|--------|------|
| 6 檔 | 3.00 / 2.00 / 1.50 / 1.20 / 1.00 / **0.85** |
| 7 檔 | 3.70 / 2.35 / 1.65 / 1.30 / 1.10 / 0.95 / **0.85** |
| 8 檔以上 | 依 7 檔，**第 8、9、10 檔均為 0.85** |

> 這些是「一定好用」的固定比值，末位三位小數都補 `000`（如 `3.000 / 2.000 / 1.500`...）。

#### 終減速比調整法（關鍵步驟）

1. **先輸入上述檔位數值**
2. **把終減速比的滑桿拉到最高（最左）**
3. 觀察此時 **理論極速**（例：325.1 km/h）
4. 將滑桿往右拉到 「**350 刻度前一格**」的位置，記錄極速變化（例：320.3 km/h）
5. 嘗試繼續往右拉一半 → 若極速突然大幅下掉（例：到 317.5），就**不要再往右**
6. 往回拉到「那半格的一半再往前一點」
7. 接下來 **每次往左撥 0.1**，觀察極速增加幅度：
   - 例：從 317.5 → 318.4（+0.9）→ 318.9（+0.5）→ 再往左 +0.3
   - **當增量小於 0.5 時就停下**，這就是該車的極速極限
8. 最終落點（範例車）：**終減速比約 3.5**

#### 依用途微調

- 跑圈圖（拉力居多）：用不到那麼高的極速 → 可**往右（加速方向）拉一點點**
- 跑公路比賽、要極速 → 按上面「小於 0.5 就停」的方法調到頂
- 漂移車：**偏加速**

> 不用追求理論極速，能達到「改到最大的那個極速」就夠了。

### 2.3 輪胎定位

#### 外傾角（Camber）——最關鍵

**基本邏輯（四驅車）**：
- **前輪**：負責拐彎 → 要**往外傾**（外八字）→ 接觸面積變大、操控變好
- **後輪**：負責跟上前輪 → 改成「正角、往內傾」→ 接觸面積變小、轉向更靈活

| 車種 | 前外傾 | 後外傾 | 邏輯 |
|------|--------|--------|------|
| 公路 | −0.5° | 0° | 前輪抓地 + 後輪保持中性，高速彎穩 |
| 拉力 | 0° | +0.5° | 後側更靈活，甩尾過彎、低速彎不會因後輪太硬推頭撞牆 |
| 漂移 | −3.5° | −2.5° | 大量負傾角配合持續滑行 |

> ⚠️ **不要反過來改**：若改成前 0 / 後 −0.5，車感會變得像前驅車，非常難開。
>
> 規律：**公路 0.5 / 0、拉力 0 / 0.5**，記住就好。

#### 束角（Toe）

- **公路 / 拉力：不改（保持 0）**，因為動了束角**一定會影響速度**。
- **漂移：均 4.0**，是唯一會動束角的場合。

#### 前後傾角（Caster）

- **三類車一律 7°**（公式圖上都標 7°）
- 拉高 → 車輛穩定、好開、反應快
- 拉低 → 反應慢、車子像鈍鈍的
- 「一般來說沒什麼缺點」，所以直接頂到最高。

### 2.4 防傾桿（Anti-roll Bar）——平衡轉向過度 / 不足

#### 兩種手感

- **轉向過度**（Oversteer）：車太能拐，後輪甩出去
- **轉向不足**（Understeer / 推頭）：車拐不過去，往外推

#### 公路 / 拉力的公式值

- **前側 1、後側 65**（前軟後硬到極致）
- 若操控過於靈活 → **前側拉硬、後側拉軟**，往中間靠，但**總體仍保持前軟後硬**

#### 調整原則（詳細）

| 症狀 | 處理方式 |
|------|---------|
| 轉向過度（太能拐） | **前硬一點、後軟一點**，往中間靠 |
| 轉向不足（推頭） | 先套「前 1 / 後 65」，若仍推頭 → 改彈簧（見 §2.5） |

> **禁忌**：不要「前硬後軟」。保持 **前軟後硬**，並讓前後比值儘量接近（約 **70 ~ 75 %**），不要讓兩者交叉。

### 2.5 彈簧（Springs）

#### 公路車的公式比例

- **前後比例 0.75 : 1**
- 實際套車時 **稍微再硬一點**，例如 **140 / 100** 或 **約 1.4 / 1.0** 的感覺

#### 拉力 / 越野

- **前後軟硬相同**，取兩側最小值中的較大值
- 範例：前最軟 50、後最軟 56 → **前後均設為 56**

#### 推頭時的彈簧處理（核心口訣）

| 狀況 | 動作 |
|------|------|
| 低速彎拐不過去（推頭） | **整體拉軟** |
| 高速彎拐不過去 | **整體拉硬** |
| 高速彎太靈敏、容易抖 / 失控 | 也是「整體拉硬」方向（但不要太硬） |

**通用口訣**：
- **越軟 → 低速彎越好**
- **越硬 → 高速彎越好**

#### 優先改硬的理由

- 高速彎優化後，低速彎的推頭可以**靠鬆油門 / 輕給油**來緩解
- 因為推頭的根本原因是「車還有向前的慣性」；沒動力時轉向自然靈敏
- 所以常見做法：**先硬 → 低速彎用油門控制**

### 2.6 車身高度（Ride Height）

| 車種 | 設定 |
|------|------|
| 公路賽 / 漂移 | **最低** |
| 拉力 / 越野 | **最高** |

**例外情況**：有些車最低高度前後懸殊（例：Apollo 前 8 cm / 後 11 cm）
- 車感覺不穩 → 把前側拉高一點，**儘量讓前後等高**，至少保證穩定
- 懶得調 → 就照上表二擇一：公路 / 漂移最低；拉力 / 越野最高

### 2.7 阻尼（Damping）

#### 公路車（作者自訂公式）

| 參數 | 前 | 後 |
|------|----|----|
| 回彈硬度 | **17** | **13** |
| 衝擊硬度 | **4** | **7** |

- 套用後的感受：**車輛在小坑小洼區域比較不掉速**，改完車身高度後，過震動路面也比較不容易彈起來、整體穩定。
- 可微調：衝擊硬度可改硬到「5 / 8」，或整體「上下加 1 都可以」，但作者一般直接套、不動。

#### 拉力 / 越野

- **回彈硬度**：前後均為 **10**
- **衝擊硬度**：均為 **2 或 2.5**
- 好記直接套。

### 2.8 空氣動力（下壓力）

#### 公路 / 拉力 / 越野

- **看需求與車輛改裝重點**：

| 車輛類型 | 策略 |
|---------|------|
| 操控優先的車 | 下壓力 **最大**（犧牲極速，換穩定） |
| 速度優先的車 | 下壓力 **最小**（留住極速） |
| 不需要壓 PI 分 | **完全不裝空力** |

#### 漂移

- **前空力拉到最右（最大）**
- **後空力拉到最左（最小）**

#### 範例（公路車）

- 某車不加空力時極速 319.3
- 空力拉到最高時極速掉到 308
- 找一個「你能接受的操控 / 極速平衡點」即可。

> **不是為了壓分就別裝空力**，很多車純粹為了留 PI 額度去裝空力，沒其他理由。

### 2.9 煞車（Brakes）

| 車種 | 建議 |
|------|------|
| 公路 / 拉力 / 越野 | **一律不改**，改了反而**爆死（撞車）機率變大** |
| 漂移 | 壓力可依手感改為 **100 ~ 110** |

> 「會改再改，不會改就別動」。

### 2.10 差速器（Differential）

- **會改的玩家自己改，不會改就套公式**

| 車種 | 中心分動 | 前加速 / 減速 | 後加速 / 減速 |
|------|---------|--------------|--------------|
| 公路（AWD） | 不變 | **90 % / 0 %** | **90 % / 0 %** |
| 拉力 / 越野 | 不變 | **100 % / 0 %** | **100 % / 0 %** |
| 漂移 | **70 % 左右** | **100 % / 0 %** | **100 % / 0 %** |

#### 關於漂移車的差速器

- 套公式後**能飄**，但若精通漂移就會知道：
  - **加速 / 減速並不會真的用這麼極端的 100 / 0**
  - 尤其是減速端，0 % 會讓車子不太好飄
  - 高手會依個人手感微調
- 作者不建議新手自行亂改漂移差速器。

## 三、漂移車的「重量比套用公式」

漂移車的防傾桿 / 彈簧 / 阻尼較特殊，不是固定值，而是依**車輛前後重量比**推算。

### 步驟

1. **進入選車界面**查看車輛前後重量比（例：前 48 % / 後 52 %）
2. **使用公式**：
   ```
   (最大值 − 最小值) × 重量比 + 最小值
   ```
3. **套用規則**：
   - 要改「**前側**」的數值 → 用**前側**重量比
   - 要改「**後側**」的數值 → 用**後側**重量比

### 套用範例

假設某車前重量比 48 %，某滑桿最小 10、最大 60：

- 前側數值 = `(60 − 10) × 0.48 + 10 = 34`
- 後側數值 = `(60 − 10) × 0.52 + 10 = 36`

同一條公式套給防傾桿、彈簧、阻尼的前後兩側滑桿。

## 四、總結與練車建議

### 套公式流程（新手版）

1. 選定車種（公路 / 拉力 / 漂移）
2. 根據 §1 的對應公式填入數值（胎壓、齒比、外傾角、防傾桿、彈簧、車高、阻尼、差速器）
3. 用 §2.2 的方法調整終減速比
4. 空氣動力依操控 / 極速需求二擇一（漂移：前大後小）
5. 煞車 / 束角一律不動（漂移除外）
6. 試跑，若出現推頭（低速彎 / 高速彎拐不過），依 §2.5 的口訣調整彈簧軟硬

### 練車建議

- 改車本身 **花不了多少 CR**，而且改裝越改越熟練，後期幾分鐘就能粗調好一輛車。
- 可以挑一些 meta 車來練手，例如：
  - **Dodge Viper ACR**（送的操控車）
  - **Lamborghini 小牛**
  - 其他常見公路車、拉力車
- 流程：**自己改一版 → 下載傳奇調校師的版本對比 → 對照手感與檔位差異 → 再回頭調自己的 → 趨近強者的設定**
- 這個過程就是在鍛鍊調校能力，不要嫌麻煩。
