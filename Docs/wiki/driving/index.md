# 🏁 駕駛操作

> **比喻**：好的駕駛像**跳舞**——你必須**預判節奏**（彎道）、**找到正確的腳位**（賽車線）、**讓身體微微前傾**（healthy oversteer）才走得最快。**車本身只是一半**，**怎麼開**是另一半。

走線、煞車、過彎、漂移、起步等駕駛技巧。

---

## 🎯 駕駛核心觀念：一句話 + 比喻

| 主題 | 控制什麼 | 生活比喻 |
|------|---------|---------|
| **Racing Line（賽車線）** | 過彎最快路徑 | 像滑雪選手選最快下山的路——不是最短，是**最能維持速度** |
| **Apex（頂點）** | 過彎最內側那個點 | 像**綁鞋帶的中間結**——前半收緊（減速）、後半放開（加速）|
| **Late Apex（晚 apex）** | 把 apex 往後挪 | 像**長直線前**晚一點轉——換取**更早全油門** |
| **Healthy Oversteer** | 健康的微甩尾 | 像滑冰選手**輕微側身**過彎——**反而更快** |
| **Slip Angle（滑動角）** | 輪胎方向與行進方向的微差 | 不是完全咬地、也不是大滑——**剛好臨界**才最快 |
| **Steering Angle（轉向角）** | 車輪能轉多大 | 並**不是越大越能轉**——只在髮夾彎與漂移反打才有用 |

> 詳見 [賽車線與彎道基礎](./賽車線與彎道基礎.md)。

---

## 🩺 快查：症狀 → 駕駛技巧調整

| 症狀 | 駕駛技巧對策 |
|------|-------------|
| **彎中常衝出去** | **晚一點煞車**前，**先晚一點轉向**——Forza 賽車線提示偏保守 |
| **出彎踩油就甩** | 練**油門控制**——RWD 出彎大多是**腳的問題**而非調校問題 |
| **長直線後彎太慢** | 試 **Late Apex** 走線——讓出彎能更早全油門 |
| **連續彎節奏抓不到** | 用**正常 Apex**（不要 late）維持節奏 |
| **髮夾彎卡住** | 看 [Steering Angle](./賽車線與彎道基礎.md#五steering-angle轉向角度) |
| **車不夠靈活** | 接受 **healthy oversteer**——不要把車調太穩 |

## 🛤️ 練車流程（HokiHoshi 推薦）

1. **第一次跑陌生賽道**：跟著 Forza 賽車線**保守**跑 3~5 圈
2. **晚一點煞、晚一點轉**——逐圈把賽車線「往後拉」
3. **找出每個彎的 Apex**——刻意瞄準
4. **接長直線的彎**——試 **Late Apex**
5. **觀察車況**：哪段 understeer / oversteer 嚴重 → 對應調校（[三段彎道診斷](../tuning/三段彎道診斷.md)）
