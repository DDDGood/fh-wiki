# ⚙️ 車輛改裝

配件升級選擇與 PI 性價比思路。

## 本分類所有主題

- [改造選擇](./改造選擇.md) — 置換引擎、傳動方式、進氣（渦輪種類）、寬體改造
- [空力與外觀配件](./空力與外觀配件.md) — 保險桿、尾翼、擾流板、引擎蓋、側翼
- [輪胎配件](./輪胎配件.md) — 輪胎材料、胎寬、輪轂、輪距
- [底盤配件](./底盤配件.md) — 煞車、懸掛、防傾桿、防滾架、減重
- [引擎配件](./引擎配件.md) — 動力配件三類、凸輪軸與飛輪
- [傳動配件](./傳動配件.md) — 離合器、變速箱（檔位數）、傳動軸、差速器

## 專題

- [PI 性價比思路](./PI性價比思路.md) ⭐ — 硬核 PI 優化視角：引擎列表、子系統排序、改裝順序

## 改裝順序建議

由下往上：**改造 → 外觀 → 輪胎 → 傳動系統 → 底盤 → 引擎**
