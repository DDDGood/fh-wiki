# ⚙️ 車輛改裝

> **比喻**：改裝像幫車**配裝備**——RPG 角色升級時要選**武器、防具、飾品**，每樣裝備都消耗有限的「**裝備格（PI）**」。**改裝就是在 PI 預算內配出最強組合**。
>
> 與 [調校](../tuning/) 的差別：**改裝換零件（影響 PI）**，**調校設參數（不影響 PI）**。

配件升級選擇與 PI 性價比思路。

---

## 🎮 6 大改裝區塊（依遊戲面板順序）

> 遊戲改裝面板由左到右順序，每個區塊附**比喻**＋**主要內容**：

### 1️⃣ [改造選擇](./改造選擇.md) — Conversions
❤️ **像心臟移植與骨架重塑**：根本性改變車的本質
**內容**：置換引擎、傳動方式、進氣（渦輪種類）、寬體改造
**新手通則**：「**想要 V8 就買原廠 V8 的車**」——通常保留原廠最穩

### 2️⃣ [空力與外觀配件](./空力與外觀配件.md) — Aero & Appearance
✈️ **像戰鬥機翅膀反過來裝**：把車「壓黏」在地上
**內容**：保險桿、尾翼、擾流板、引擎蓋、側翼
**簡明法則**：**降 drag 一律拿、加 drag 不裝**；可調前 aero 是車上**最強升級之一**

### 3️⃣ [輪胎配件](./輪胎配件.md) — Tires & Rims
👟 **像選一雙鞋**：車與地面**唯一**的接觸點，影響最直接
**內容**：胎質、胎寬、輪轂、輪距
**2025 顛覆觀點**：**Sport 胎避免**、**B–S1 公路用 Rally / Drift 胎**、**前胎寬保留原廠**

### 4️⃣ [傳動配件](./傳動配件.md) — Drivetrain
💪 **像從心臟到腳的筋骨**：把引擎力量送到輪上
**內容**：離合器、變速箱（檔位數）、傳動軸、差速器
**鐵律**：**Sport diff 絕對不要用**——同樣 PI 但**沒任何性能**

### 5️⃣ [底盤配件](./底盤配件.md) — Platform & Handling
🦴 **像骨架與感測神經**：決定車怎麼承受路面與動作
**內容**：煞車、懸吊、防傾桿、防滾架、減重
**⚠️ 警告**：**升懸吊會移除原廠四輪轉向（4WS）**——少數車（NSX、部分 Lambo / Porsche）需保留原廠

### 6️⃣ [引擎配件](./引擎配件.md) — Engine
❤️ **像心臟強化手術**：直接給車**更多馬力**
**內容**：進氣、排氣、燃料、點火、增壓、凸輪、飛輪、油冷／中冷
**鐵律**：**先解鎖調校項目，剩餘 PI 才砸引擎**

## 🩺 快查：你的 build 想幹嘛？→ 重點升級什麼

### 想要極速（直線快）
重點：**引擎大馬力 + 減 drag**
1. [引擎](./引擎配件.md)：排氣／進氣／排量
2. [改造](./改造選擇.md)：引擎置換、強增壓
3. [底盤](./底盤配件.md)：減重
4. [空力](./空力與外觀配件.md)：**不要加 drag**，可調後翼若降 PI 才裝

### 想要過彎抓地（圈速快）
重點：**輪胎 + 懸吊 + 空力**
1. [輪胎](./輪胎配件.md)：升級胎質（HokiHoshi 2025：B–S1 公路用 **Rally / Drift 胎**）
2. [底盤](./底盤配件.md)：race 懸吊、ARB 全升、減重
3. [空力](./空力與外觀配件.md)：**可調前 aero**（最強升級之一）
4. [傳動](./傳動配件.md)：差速器（必升、絕不用 Sport）

### 想要全能 build
重點：**胎與懸吊優先、剩餘 PI 馬力與減重二擇一**
1. 解鎖調校項目（懸吊、ARB、diff）
2. 配置好胎與變速箱
3. 必要時加可調 aero、煞車
4. 剩餘 PI **min-max**：要嘛**全減重**、要嘛**全加馬力**

> 經典金句（HokiHoshi 2021）：「**加馬力讓你直線更快；減重讓你哪裡都更快**」

### 想壓 PI 上限（剛好不超 PI）
微調工具（幾乎不影響性能）：
- [輪胎](./輪胎配件.md) 輪圈樣式／尺寸（後輪圈加大常降 PI）
- [傳動](./傳動配件.md) 傳動軸、變速箱
- [空力](./空力與外觀配件.md) 不可調車身件、可調後翼
- [引擎](./引擎配件.md) 飛輪
詳見 [PI 性價比思路](./PI性價比思路.md)。

## 🚗 各車種的改裝重點

### 公路車（Road / Track）
1. **輪胎**：原廠／街車（低級距）／**Rally / Drift（B-S1 級）**
2. **底盤**：race 懸吊、ARB 升頂、減重
3. **空力**：可調前 aero（**A 級以上必裝**）；後翼依 PI 餘裕
4. **引擎**：剩餘 PI 砸這裡
5. **驅動**：保留原廠或改 AWD

### 拉力／越野車
1. **輪胎**：**Off-road race 胎**
2. **底盤**：**rally 懸吊**、底盤強化（老車）
3. **驅動**：**AWD**
4. **避免**：升級煞車（HokiHoshi：「越野拉力完全不用」）

### 漂移車
1. **驅動**：**RWD**（不換）
2. **輪胎**：**漂移胎**
3. **底盤**：**漂移懸吊**
4. **傳動**：**Race transmission**（解鎖個別檔位）+ Race / Rally diff
5. 煞車解鎖煞車調校（漂移控制用）

### 直線加速（Drag）
1. **引擎**：**全力堆馬力**
2. **底盤**：**最大減重**
3. **輪胎**：**Drag 胎**或前後不對稱配置
4. **空力**：**不裝**（避免 drag）
5. **傳動**：適合的變速箱

---

## ⚠️ 三大「踩雷警告」

1. **⚠️ 升懸吊會移除原廠四輪轉向**——少數車（NSX、部分 Lambo / Porsche）原廠有 4WS，升級懸吊**會失去**
2. **⚠️ 寬體可能擋住可調前 aero**——可調前 aero 是車上最強升級之一，安裝寬體前先確認
3. **⚠️ 絕對不用 Sport diff**——同樣 PI 但**沒任何性能**

---

## 🔧 改裝順序

依遊戲面板由左到右**從根本到細節**：
**改造（1）→ 外觀（2）→ 輪胎（3）→ 傳動（4）→ 底盤（5）→ 引擎（6）**

> 為何這個順序：先決定根本（改造）→ 設定**空氣動力**與**接地**→ 動力**傳輸**→ 然後**操控與煞停**→ 最後**動力**填 PI 餘裕。
>
> HokiHoshi 強調：**先解鎖調校項目**（懸吊、ARB、diff），**剩餘 PI 才砸引擎**。
