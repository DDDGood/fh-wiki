# 輪胎配件選擇

---

## 輪胎材料

| 類型 | 特性 |
|------|------|
| **原廠胎** | 多數車原裝為公路車胎 |
| **街車胎 / 跑車胎** | 一般不常用 |
| **半熱熔胎** | 三種花紋相同、性能相同 |
| **熱熔胎**（光溜溜） | 遊戲內抓地最強。四種帶「熱熔」的輪胎會在胎溫升高時微微提升抓地，多見於高等級賽車 |
| **漂移胎** | 漂移車專用，也常用於後驅車 |
| **拉力胎** | 作者常用。抓地比比賽胎稍差，但**均衡、性價比高**，多用於 A 級、S1 公路車 |
| **越野胎** | 泥地上的「熱熔胎」。越野、拉力車都推薦（遊戲設定下拉力胎跑拉力賽反而不如越野胎） |
| **雪胎** | 越野胎的青春版。對積雪、風火輪、冰雪路段有奇效，局限性大，一般不推薦 |
| **直線加速胎** | 專用於直線加速賽，對 S2 降級車奇效，乾燥路面可嘗試 |
| **懷舊賽車胎** | 比街車胎好很多 |
| **復古白圈胎** | 少見，但給 B 級以下賽車安裝比較有性價比 |

### 個人分類

- **公路車胎**：街車、跑車、半熱熔、熱熔、漂移、拉力、直線加速、懷舊、復古
- **泥地車胎**：拉力、越野、雪胎
- **漂移車胎**：漂移、越野

### 硬核版：依車種的策略選擇

| 車種 | 建議胎種 | 理由 |
|------|---------|------|
| **公路調校** | **拉力胎** | 不同路面與天氣對其性能影響最小 |
| **拉力 / 越野調校** | **越野胎** | 直接越野胎最穩 |

## 胎寬

### 前輪胎寬

- 前輪為**轉向輪**，增加胎寬可提升轉向靈活性與抓地力。
- **前驅、四驅車**：增大前胎寬可提升動力輸出與加速。
- 但意味著更重、更高阻力、佔用更多性能分，非剛需請謹慎。

### 後輪胎寬

- **後驅、四驅車**：增大後胎寬可明顯改善抓地與加速，**後驅車尤甚**。
- **前驅車**：後胎寬增大會導致轉向遲鈍、阻力增加，謹慎選擇。

---

## 輪轂

### 樣式

- 不同輪轂重量不同，選擇時注意重量影響。
- 若需**控分**，可適當選重一點的輪轂。

### 尺寸

- 越大 → **越重**、胎壁越小。
- **前輪**尺寸越大可略提升操控性；但對越野車可能更容易打滑。
- **後輪**輪轂尺寸原理同前輪；因後輪不負責轉向，主要差別在重量，可藉由**加大後輪輪轂達到降分效果**。

### 硬核版：輪圈 PI 思路

- 輪圈款式與大小主要用來**配重**
- **後輪圈加大**通常能有效**降低 PI**
- **前輪圈**因車而異：有些車前輪圈加大會**同時增重並增加 PI**，要看

---

## 前後輪距

- 越寬，**側向穩定性**越強。
- **前輪距**增加可改善轉向穩定性，不考慮壓分時**推薦拉滿**。
- **後輪距**增加同理提升側向穩定性，過彎路段、複雜路面越高越穩。雖後輪距高可能導致些微轉向不足，但影響可忽略，**推薦拉滿**。
