# 傳動系統配件

> **💪 用比喻理解**：傳動是**從心臟到腳**的「**筋骨**」——把引擎的力量送到輪上：
>
> - **離合器**：心臟與筋的**閘門**——換檔時的**動力空窗時間**長短由它決定
> - **變速箱**：**可變比例齒輪**——同一份馬力可以「**省力慢走（低檔）**」或「**用力快跑（高檔）**」
> - **傳動軸（Driveshaft）**：**主筋**——主要提供**減重**與微調 PI
> - **差速器**：**膝蓋彎度限制**——左右輪可以多不同步（過彎用）
>
> 一個重要鐵律：**Sport diff 絕對不要用**——同樣 PI 但**沒任何性能**。一律升到 race / rally / off-road / drift 對應 build 類型的版本。

---

## 離合器

- 地平線世界觀基本沒有 CVT 無段變速，**自排也會有換檔動作**。
- 離合器可**縮短換檔時間** → 減少動力空窗。
- 不同離合器縮短的時間與減重效果不同，佔用性能分較多。
- **檔位多、操控車**需搭配更高級的離合器。
- **S2 級的離合器**基本只提供減重，按需取捨。

### 硬核補充

- 升級離合器能改善換檔速度，但 **PI 性價比極低**
- **使用手動離合時不要升級離合器**

### HokiHoshi 2021：兩個觀察點

1. **老車慢車升級非常有用**：尤其搭配 Auto / Manual Clutchless 模式時——老車原廠換檔慢，升級離合器收益明顯
2. **升級時關注**「**換檔時間（shift time）**」**的變化** —— 這是離合器升級最直觀的數據反饋，比單看 PI 更實用

---

## 變速箱

幾乎每一台車都需要升級變速箱。可提供合適轉速區間、加快換檔、少量減重。

### 分類

| 類型 | 可調項目 |
|------|----------|
| 未解鎖調整 | — |
| 解鎖最終傳動比 | 僅最終傳動比 |
| **賽車變速箱** | 全部傳動比（**首選**） |

### 檔位數

- 賽車變速箱後的極速數字代表檔位數。例如「極速」= 9 檔；若未寫通常是 **6 檔**。
- **漂移變速箱** 為 **4 檔**，一般不選。

### 性能分不足時

退而求其次：選**更多檔但更重**的變速箱，或直接選**跑車變速箱**；只要能調校就有提升空間。

### 硬核補充：變速箱是增重與卡 PI 的重要手段

- 要改賽車變速箱 → 直接套用**十速變速箱**；有 PI 空間再選檔位較少的
- 部分**老爺車 / 復古車**變速箱升級消耗 PI 極大 → **沿用原廠**
- 有些車原廠齒輪比就很完美，原廠即可
- 有些車升級街車或跑車變速箱反而**能減 PI**，要仔細比對

### HokiHoshi 2025：「7+ 速幾乎不值」（與硬核並列觀點）

> HokiHoshi 主張「**升級到超過 7 速幾乎永遠不值得**」——除非原廠齒比不佳或只有 4~5 速。

兩派觀點並存：
- **硬核派**：直接套用十速作為**配重 / 卡 PI** 工具（透過最終傳動比把十速當 6 或 7 檔用）
- **HokiHoshi 派**：純從性能角度看，超過 7 速沒收益，配重請另覓他法

> 結論：若**為了配重 / 卡 PI**，硬核派的十速可行；若**單純追求性能**，6~7 檔賽車變速箱已足夠。

## 傳動軸

- 有多種不同重量傳動軸，主要提供**減重**。
- 可能造成加速略慢，但**可忽略**。
- 性能分充足時**建議安裝**。

---

## 差速器

- 升級**不改變硬件**，但**解鎖差速器調校**。
- **每台車必備**。
- 一般**不選跑車差速器**；其他類型差速器調校效果一樣，選對應類型即可。

### 硬核補充

**雙向可調差速器必裝**。不同類型的雙向差速器僅初始參數不同（差速器調校效果一樣）。

### ⚠️ HokiHoshi 2025：絕對不用 Sport diff

> HokiHoshi 給的明確警告：「**Never use the sport diff.**」

- **永遠升級**——免費，且解鎖 diff 調校
- **絕對不要用 Sport diff**（與既有「一般不選跑車差速器」立場一致但更明確的禁令）
- 依 build 類型升到對應的 race / rally / off-road / drift diff

### HokiHoshi 2021：差速器類型細分

> 同樣選對應類型 diff，但 HokiHoshi 2021 提供更細的應用區分：

| Diff 類型 | 適用 |
|-----------|------|
| **Race diff** | 公路 build（一般 race / track 用）|
| **Rally diff** | Rally 與 Off-road build 通用 |
| **Off-road diff** | **只用在純越野 build**（沙丘越野車、攀岩車）——不要無腦用在一般 rally |
| **Drift diff** | 漂移 build |

> **Off-road diff 是 niche 選擇**——除非你做的是「純越野」build，否則 rally diff 已夠用。

## 傳動軸升級

主要用來**增重**，需要配重時再考慮。
