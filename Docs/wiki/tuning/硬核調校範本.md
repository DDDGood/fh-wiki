# 硬核調校範本

> 本文是《硬核調校指南》的**完整調校參數矩陣**，可直接套用。原作風格：**不要害怕走極端**。

與其他速查／參考檔的關係：
- [三車種公式速查](./三車種公式速查.md) — 第九期版的三車種範本
- [QuickTune 基準值總表](./QuickTune基準值總表.md) — QuickTune 的完整矩陣
- **本檔** — 硬核版的完整矩陣（第 4 套說法）

## 胎壓（依車重分級）

| 調校類型 | 車重 | 前後胎壓 |
|---|---|---|
| 公路賽 | < 1000 kg | 前後 **1.5 磅** |
| 公路賽 | 1000–1400 kg | 前後 **1.8 磅** |
| 公路賽 | > 1400 kg | 前後 **2.0 磅** |
| 拉力 / 越野 | — | 在公路基礎上 **−0.3** |
| 直線加速 | — | **前最高、後最低** |
| 漂移 | — | 通常**前高後低**，具體因車與習慣 |

> 比賽中輪胎會升溫達到理想胎壓。

---

## 齒輪比

- 目標：從二檔開始，**每一檔最高轉速落在理想功率帶內**；每檔起始轉速依序遞增。
- 最終傳動比：讓車輛在**最後一檔**拉到功率峰值時剛好達到極速（末端頂到右上角）。
- **十速變速箱**不代表要用十個檔 — 透過最終傳動比當**六檔或七檔**用。
- 起步/出彎加速打滑：把低檔拉長（賽車變速箱調第一檔；跑車變速箱調最終傳動比）。

## 輪胎定位（完整表）

| 調校類型 | 前外傾 | 後外傾 | 前束 | 後束 | 後傾角 |
|---|---|---|---|---|---|
| **公路** | −0.5 | −0.5 | 0.3 | 0.1 | **7** |
| **拉力 / 越野** | 0 | −0.5 | 0 | 0 | **7** |
| **漂移** | −（負外傾）| −（負外傾）| **2** | **1** | **7** |
| **直線加速**（含翹頭、翹屁股衍生）| 0 | 0 | 0 | 0 | **7** |

> **後傾角一律拉滿至 7**，若轉向太靈活最後再微調。

前驅/四驅（天生推頭）：前輪束角加不超過 **+0.5**，後輪束角不超過 **+0.2**。
大馬力後驅：後輪加一點**負束角**（提升後輪穩定性）。

## 防傾桿

| 調校類型 | 前防傾桿 | 後防傾桿 |
|---|---|---|
| 前驅 / 四驅（基本值） | **1** | **65** |
| 四驅（改善彎中穩定性） | **10** | **6.5** |
| 前驅（改善彎中穩定性） | **1** | **5.6** |
| 後驅（初始） | **10** | **5.6** |
| 拉力 / 越野（初始） | **10** | **2.5** |
| 大皮卡（如 Mercedes X-Class）| **1** | **最軟** |
| 直線加速 | **65** | **1** |

原則：前軟後硬 → 改善推頭；前硬後軟 → 改善甩尾。**前防傾桿尤其重要**。

## 彈簧

| 驅動方式 | 前彈簧 | 後彈簧 |
|---|---|---|
| **前驅 / 四驅** | 最軟 | 約在 **2/3** 位置 |
| **四驅（另一思路）** | 最軟 | 最硬 |
| **後驅** | 不超過調整範圍的 1/3，**初始約 80** | **初始約 100**（依試車調整） |
| **大馬力後驅** | 前後可一樣，甚至都用最軟 | 同左 |
| **拉力懸掛（共通）** | 不超過行程的 **1/6** | 不超過 **1/4** |
| **拉力調校** | 偏軟 | 偏硬（前軟後硬提升轉向） |
| **越野調校** | 前後軟硬度**一樣**（起跳時不推頭不翹頭） | 同左 |
| **輕量越野車** | 前後都最軟 | 同左 |
| **直線加速** | 最硬 | 最軟 |

> ⚠️ 遊戲早期流傳的「**四代彈簧計算公式**」計算出來的結果有**嚴重轉向不足**，**不推薦使用**。

## 車身高度

| 調校類型 | 設定 |
|---|---|
| **公路**（一般） | 用**預設數值**，以「高的一側」配平 |
| **公路特例**（KSG CCGT、法拉利 FXX 等）| 可**拉高車頭**，提高操控可控性，甚至提升極速 |
| **拉力 / 越野** | 前後都**最高** |
| **直線加速** | 前輪**最高**、後輪**最低** |
| **漂移** | 前後**統一最低** |

## 阻尼（公式法）

| 項目 | 建議 |
|---|---|
| 回彈一般範圍 | **8 ~ 14** |
| 壓縮（拉力 / 越野）| 回彈 × **0.25** |
| 壓縮（公路）| 回彈 × **0.4** |

**原理**：
- 回彈速度永遠大於壓縮速度
- 理論上壓縮 = 回彈的 50~75%，但地平線 5 實測偏軟更好（**壓縮越軟，吸震越強**）
- 高級懸掛預設 63%，**實測太硬不可取**

### 症狀對照

| 症狀 | 調整 |
|---|---|
| 剎車或落地後常被彈飛失控 | **回彈調硬** |
| 過路肩時輪胎彈起 | **壓縮硬度調軟** |
| 改善轉向不足 | 回彈**前軟後硬** |
| 改善轉向過度 | 回彈**前硬後軟** |
| 前驅 / 四驅的一般作法 | 範圍內**前軟後硬** |
| **直線加速** | **前 20 / 後 1** |

## 空氣動力（下壓力）

| 驅動方式 | 前下壓力 | 後下壓力 |
|---|---|---|
| **前驅 / 四驅** | **拉到最高** | 不超過**一半**位置 |
| **後驅**（A 級以下）| **拉滿** | 先視為**最低**，再依試車逐步調大 |
| **後驅**（S1 以上大馬力）| 拉滿 | 依需求偏大，甚至**拉滿** |
| **直線加速** | — | **不需要**升級空力套件 |

原則：
- 後下壓力越大 → 後輪抓地力越強，但太大會造成**轉向不足**
- PI 等級越高 → 後下壓力需求越大

---

## 剎車

| 項目 | 設定 |
|---|---|
| **剎車平衡**（一般） | **> 50%**（偏前增轉向過度；偏後減轉向過度、增轉向不足） |
| **剎車平衡**（特例：第六元素特別版、CCGT 等天生轉向靈活的車） | 可設 **< 50%** 以提升剎車穩定性 |
| **制動力（手柄 / 方向盤，無 ABS）後驅** | **60 ~ 80%** |
| **制動力（鍵盤後驅）** | **60 ~ 80%** |
| **制動力（開 ABS）** | **80 ~ 200%** 都可，比較靈活 |

- **制動力低** → 增加剎車動作行程，利於循跡剎車與精細控制
- **制動力高** → 增加剎車力度、減少行程

## 差速器

### 初始值

| 驅動方式 | 加速 | 減速 |
|---|---|---|
| **前驅** | 50% | 0% |
| **後驅** | 80% | 5% |
| **大馬力後驅**（需穩定性） | 調**低後差加速** | 適當調**高後差減速** |

### 範圍

- **後差加速**：馬力越大需越大，一般 **50 ~ 95%**
- **前差加速**（公路）：**20 ~ 90%**
- **前差加速**（拉力）：**30 ~ 90%**
- **前差加速**（越野）：**80 ~ 100%**
- **前差減速**：一般設為 **0**；若已調高後差減速仍推頭，可稍微調高。**通常前差減速不高於前差加速**。

### 中央差速器

| 調校類型 | 中差 |
|---|---|
| 公路（四驅） | **75%** 左右 |
| 拉力 | **60%** 左右 |

### 後驅漂移常用組合
**後差加速 100% / 後差減速 90%** — 鎖得越死，車越容易飄。
