# 三車種調校數值公式速查

本表提供**公路車、拉力/越野車、漂移車**三組可直接套用的調校數值公式。

> 本速查表為**新手起手值**；各參數的詳細原理、與其他來源的數值對比，請參見對應細項文章（文末連結）。

---

## 公路車（Road）

| 項目 | 數值 |
|------|------|
| **胎壓** | 前 **1.3**、後 **1.4** |
| **6 檔齒比**（以 6.2 L V8 引擎為例）| **3.00 / 2.00 / 1.50 / 1.20 / 1.00 / 0.85** |
| **7 檔齒比** | **3.70 / 2.35 / 1.65 / 1.30 / 1.10 / 0.95 / 0.85** |
| **8 檔以上** | 依 7 檔設置，第 8、9、10 檔均為 **0.85** |
| **終減速比** | 自行調整（見[齒比](./齒比.md)的終減速比調整法）|
| **外傾角** | 前 **−0.5°**、後 **0°** |
| **束角** | 不變（保持 0）|
| **前後傾角（Caster）** | **7°** |
| **防傾桿** | 前 **1** / 後 **65**（若過靈活 → 前硬後軟，但整體保持前軟後硬）|
| **彈簧** | 前後比例 **0.75 : 1**。總體偏軟 → 低速彎靈活；總體偏硬 → 高速彎靈活 |
| **阻尼：回彈硬度** | 前 **17**、後 **13** |
| **阻尼：衝擊硬度** | 前 **4**、後 **7** |
| **車身高度** | **最低** |
| **空力** | 根據車輛操控決定偏車速或偏操控 |
| **煞車** | 不進行改動 |
| **差速器** | 前 **90 % / 0 %**、後 **90 % / 0 %**、中心不變 |

## 拉力 / 越野車（Rally / Offroad）

> 原圖右上角標註：數據來源 **@Rock**（UP 主引用他人資料）

| 項目 | 數值 |
|------|------|
| **胎壓** | 前後 **1.0** |
| **齒輪設備** | ▸ 跑圈圖 → 最終傳動比向**加速**方向拉<br>▸ 跑衝刺賽 → 向**極速**方向拉<br>▸ 齒比設定參考公路；依車輛引擎微調 |
| **外傾角** | 前 **0°**、後 **+0.5°** |
| **束角** | 不變（保持 0）|
| **前後傾角（Caster）** | **7°** |
| **防傾桿** | 前 **1** / 後 **65**（同公路）|
| **彈簧** | 前後軟硬相同，**取兩側最小值中的較大值**<br>範例：前最軟 50、後最軟 56 → 前後均設為 **56** |
| **阻尼：回彈硬度** | 前後均 **10** |
| **阻尼：衝擊硬度** | 均 **2 或 2.5** |
| **車身高度** | **最高** |
| **空力** | 根據車輛操控決定偏車速或偏操控 |
| **煞車** | 不進行改動 |
| **差速器** | 前 **100 % / 0 %**、後 **100 % / 0 %**、中心不變 |

## 漂移車（Drift）

> 原圖右上角標註：數據來源 **@SSSSSSwitch**（漂移專業 UP 主）

| 項目 | 數值 |
|------|------|
| **胎壓** | 維持原數值即可 |
| **齒輪設備** | ▸ 最終傳動比偏**加速**<br>▸ 齒比優先保證前 5 檔<br>▸ 需根據實際試車決定 |
| **齒比試用值**（不知如何試車先套）| **4.15 / 2.90 / 2.05 / 1.65 / 1.40**（前 5 檔）後續依加速/車速平衡 |
| **外傾角** | 前 **−3.5°**、後 **−2.5°** |
| **束角** | 均 **4.0** |
| **前後傾角（Caster）** | **7°** |
| **防傾桿 / 彈簧 / 阻尼** | **依重量比公式推算**（見[漂移重量比公式](./漂移重量比公式.md)）|
| **車身高度** | 均 **最低** |
| **空力** | 前空力拉**最右**、後空力拉**最左** |
| **煞車** | 壓力可依手感改為 **100 ~ 110** |
| **差速器** | 前 **100 % / 0 %**、後 **100 % / 0 %**、中心 **70 %** 左右 |

> 套公式後能飄，但精通漂移者會知道「加速 / 減速並非真的 100 / 0」，會依個人手感微調。

## 使用指引（新手版）

### 套公式流程

1. 選定車種（公路 / 拉力 / 漂移）
2. 按上表填入數值（胎壓、齒比、外傾角、防傾桿、彈簧、車高、阻尼、差速器）
3. 用[齒比](./齒比.md)的終減速比調整法調整終減速比
4. 空氣動力依操控 / 極速需求二擇一（漂移：前大後小）
5. 煞車 / 束角一律不動（漂移除外）
6. 試跑，若出現推頭（低速彎 / 高速彎拐不過），依[彈簧與車高](./彈簧與車高.md)的推頭口訣調整彈簧軟硬

### 練車建議

- 改車本身**花不了多少 CR**，而且改裝越改越熟練，後期幾分鐘就能粗調好一輛車
- 可以挑一些 meta 車來練手，例如：
  - **Dodge Viper ACR**（送的操控車）
  - **Lamborghini 小牛**
  - 其他常見公路車、拉力車
- 流程：**自己改一版 → 下載傳奇調校師的版本對比 → 對照手感與檔位差異 → 再回頭調自己的 → 趨近強者的設定**
