# 四輪定位

> **👟 用比喻理解**：定位像**輪胎與地面的「站姿」**——
> - **外傾角（Camber）**：像**穿高跟鞋**——鞋底傾斜，**站直時只有外緣著地，但傾倒過彎時整個鞋底壓滿**。負外傾就是讓輪胎在過彎傾倒時「**整個胎面壓在地上**」。
> - **束角（Toe）**：像**走路的腳尖方向**——**內八（toe in）**穩、不易滑；**外八（toe out）**靈活、好轉但會慢
> - **後傾角（Caster）**：像**超市購物車的輪子**——**回正能力**。值越大車越自動回正（穩、好開、反應快）；越小越鈍。**進彎時還會自動加深 camber 角度**。
>
> **記憶技巧**：camber 看正面、toe 看上面、caster 看側面。

> **修訂 2026-04-21**：從《萌新攻略第九期》併列**外傾角、前後傾角（Caster）、漂移束角**三處的數值衝突；補入**前後輪物理分工**原理。

四輪定位共三項可調：**外傾角、束角、前後傾角**。

---

## 外傾角（Camber）

從車頭看：
- **八字形** → 負（外傾角）
- **內扣** → 正（內傾角）

過彎時車身會外傾，為保證外側抓地，通常設為**負值**。

### 物理分工（第九期原理）

- **前輪**：負責拐彎 → 要**往外傾**（負值、外八字）→ 接觸面積變大、操控變好
- **後輪**：負責跟上前輪 → 改成「正角、往內傾」→ 接觸面積變小、轉向更靈活

> ⚠️ **不要反過來改**：若改成前 0 / 後 −0.5，車感會變得像前驅車，非常難開。

### 推薦初始值（併列：三份來源數值不同）

| 車種 | 調校教學 | 萌新攻略第九期 | 硬核調校指南 |
|------|---------|---------------|-------------|
| 公路車 | 前 **−1.5°** / 後 **−1.3°** | 前 **−0.5°** / 後 **0°** | 前 **−0.5°** / 後 **−0.5°** |
| 拉力 / 越野車 | 前後均 **−0.5°** | 前 **0°** / 後 **+0.5°** | 前 **0°** / 後 **−0.5°** |
| 漂移車 | 先設 **−5.0°**，建議直接拉滿 | 前 **−3.5°** / 後 **−2.5°** | 前後皆負外傾（具體數值因車） |
| 直線加速 | — | — | 前後皆 **0°** |

> 第九期記法：**公路 0.5 / 0、拉力 0 / 0.5**（數值符號依物理分工）。
>
> 差異原因可能是三位作者對「前輪抓地 vs 後輪靈活」取向不同，建議都試試找合適的手感。

### 依驅動方式的 Camber 偏好（HokiHoshi 2025 維度）

> 中文來源按**車種**（公路／拉力／漂移）分，HokiHoshi 提供**按驅動方式**的另一切角，**與按車種的數值不衝突，是另一個維度**。

| 驅動 | Camber 偏好 |
|------|-------------|
| **RWD** | 前 camber **比後高**（前更負） |
| **FWD** | 後 camber **比前高**（後更負） |
| **AWD** | 前後**接近** |

> 多數車落在 **0 ~ -2°** 範圍。HokiHoshi 預設此範圍，依需要微調；真要精調需看胎溫遙測與 live camber 數據。

### 試車微調方法（兩份來源一致）

按 **T** 打開遙測，選輪胎參數，觀察**外側輪胎胎溫**（分內側、中間、外側）：

- 內側胎溫 > 外側 → 向**正**方向（調小負值）調整
- 外側胎溫 > 內側 → 繼續往**負**方向調整

## 束角（Toe）

從車頂看：
- **內扣** → 內束角（負值） → 車身穩定
- **外翻** → 外束角（正值） → 轉向靈活

原理涉及阿克曼角較複雜。動了束角**一定會影響速度**（增加阻力）。

### 建議（公路 / 拉力 / 越野）

**不改**（保持 0）— 多數來源一致（硬核的精細建議見下）。

若需改善轉向：可嘗試增大前輪外束角，**不超過 0.5**（調校教學）。

### 硬核版的精細建議

| 情境 | 前束角 | 後束角 |
|------|--------|--------|
| 前驅 / 四驅（天生推頭）| 加不超過 **+0.5** 外束角 | 加不超過 **+0.2** |
| 大馬力後驅 | — | 加一點**負束角**（提升後輪穩定性）|
| 公路（整體）| **+0.3** | **+0.1** |

> 每 **0.1** 的差距都會顯著影響加速、減速與胎溫。
> **正束角** → 輕微改善轉向不足，過大會不穩定。
> **負束角** → 讓車保持直線。

### 漂移車（併列：三份來源數值不同）

| 來源 | 漂移束角 |
|------|---------|
| 調校教學 | 前後「**拉滿（正值）**」以提高靈活性，更快進入漂移狀態 |
| 萌新攻略第九期 | 均 **4.0** |
| 硬核調校指南 | 前 **2** / 後 **1** |

> 「拉滿」的實際值依車輛而定，可能就是 4.0 上下；三家方向一致（皆正值 = 外束角），差在程度。

## 前後傾角（Caster）

類似商場手推車輪子的回正原理。

- 前後傾角越大 → 車輪回正越快
- 拉高 → 車輛穩定、好開、反應快
- 拉低 → 反應慢、車子像鈍鈍的

### 建議值（併列：三份來源描述）

| 來源 | 建議 |
|------|------|
| 調校教學 | **直接拉滿**（三車種皆然），或不動保持原廠值 |
| 萌新攻略第九期 | **三車種一律 7°**（一般來說沒什麼缺點，直接頂到最高）|
| 硬核調校指南 | 初始**直接拉到最大值 7**，若轉向太靈活最後再微調 |

> 三家一致：「拉滿 / 7°」實務上同義，直接朝頂端設定即可。
> **後傾角越小 → 前輪需要更多負外傾**（補償抓地）。
