# 遙測（Telemetry）使用指南

調校完成後要在試車中驗證並優化，**遙測是最直觀的數據來源**。

---

## 開啟方式

- **預設鍵位**：進階控制選單可將「HANNA」替換為「遙測鍵」，之後按方向鍵下即可打開。
- **自訂鍵位**：只能用鍵盤開啟。

---

## 常用頁面

### ① 一般（General）頁面

- **左側**：即時顯示發動機輸出功率、扭矩、轉速
- **右側**：油門與剎車行程
- **用途**：查看轉速、剎車抱死臨界值 → 優化[齒比](./齒比.md)、練習剎車習慣

---

### ② 懸掛系統（Suspension）頁面

- 紫粉色 = 懸掛壓縮程度，靜止為 **50%**
- 地面不顛簸時，紫粉色若把白色「**填滿**」→ 表示懸掛已壓到底
  - 解法：調硬[彈簧](./彈簧與車高.md)、調高壓縮硬度、或調高車身高度

### 懸掛行程的理想範圍

- **行駛壓縮理想範圍**：懸掛行程的 **15 ~ 85%**
- **平路上紫粉色波動幅度很大** → **彈簧調硬**
- **波動幅度偏小** → **彈簧調軟**

---

### ③ 輪胎資料（用得最多）

- 用來優化 [四輪定位](./四輪定位.md)
- 過彎時重點觀察**外側輪胎的外傾角**：
  - 理論上 **0°** 最理想
  - 但路面不平，過彎時出現**約 2°**的外傾角都算合理（輪胎會形變，2° 時實際仍緊貼地面）
  - **不用糾結**，只要不常出現**正外傾**就行

---

### ④ 輪胎溫度（Tire Temperature）頁面

即時反饋**內側 / 中間 / 外側**胎溫；輪胎形狀白框顯示各區溫度分佈。

### 胎溫色區對照

| 顏色 | 狀態 |
|---|---|
| 淺藍 | 胎溫**太冷** |
| 透明 | 胎溫**尚可** |
| 半透明 / 黃 / 淺黃 | **理想工作溫度** |
| 橙紅 | 輪胎**過熱** |

### 使用原則

- 用來驗證 [胎壓](./胎壓.md) + [四輪定位](./四輪定位.md) 的設定
- 環境溫度、天氣、路況都會影響胎溫 → 建議在**勁敵模式（Rivals）** 中測試
- **理想狀態**：
  - 比賽中輪胎**內側溫度 ≥ 中間 ≥ 外側**
  - 長時間維持在**半透明 / 黃 / 淺黃**
  - 剎車升溫後出彎能迅速降回淺黃
  - **內外溫差 ≤ 10°C**
- **溫差太大** → 負外傾角度過大 → **調小負外傾**即可
