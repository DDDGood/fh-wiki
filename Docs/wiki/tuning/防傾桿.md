# 防傾桿調校

> **🪢 用比喻理解**：防傾桿像**連接左右輪的橡皮筋**——
> - **硬橡皮筋**：左輪壓下去時**強迫右輪也跟著壓**——**整個車身不易側傾**（穩、過彎反應快但不靈動）
> - **軟橡皮筋**：左右輪可以**獨立動**——**車身可側傾**（過彎傾倒明顯、跟手感強但不穩）
>
> **前 vs 後**：
> - **前硬後軟**：前段穩、後段甩 → 易 oversteer（**禁忌**——很難開）
> - **前軟後硬**：前段靈動、後段穩 → 推頭傾向，**主流配置**
>
> 「**165 梗**」（前 1 / 後 65）= 把**前橡皮筋鬆到最軟、後橡皮筋拉到最緊** → 車前段非常靈動、後段很穩（AWD 神配置）。

> **修訂 2026-04-21**：從《萌新攻略第九期》補入**禁忌（不要前硬後軟）**、**前後比值原則（約 70~75%）** 與依症狀（轉向過度／推頭）處理的對照表。

## 原理

調整車身**側向穩定性**：

- **軟** → 過彎外傾明顯
- **硬** → 穩定但影響轉向靈活性

調好可讓車輛動態穩定、同時靈活良好。

## 前後對比效果

| 設定（前 / 後） | 效果 |
|-----------------|------|
| 前 **1** / 後 **65** | 車身容易甩尾、**靈活** |
| 前 **65** / 後 **1** | 車身**推頭**、穩定 |

## 依車種的建議初始值

| 車種 | 防傾桿（前 / 後） | 說明 |
|------|-------------------|------|
| 多數四驅車 / 公路車 / 拉力越野車 | 前 **1** / 後 **65** | 改善靈活性（多份來源一致） |
| 有後輪轉向的四驅車 | 前 **12** / 後 **48** 再調 | — |
| 後驅車（版本 A）| 先**前後 32**，再依車身動態逐步調整 | 調校教學；本就易甩尾 |

> 調校教學法不適用所有後驅車，請謹慎。

### 硬核版：進階車種細分

| 調校類型 | 前防傾桿 | 後防傾桿 |
|---------|---------|---------|
| 前驅 / 四驅（基本值） | **1** | **65** |
| 四驅（改善彎中穩定性） | **10** | **6.5** |
| 前驅（改善彎中穩定性） | **1** | **5.6** |
| 後驅（硬核版初始） | **10** | **5.6** |
| 拉力 / 越野（初始） | **10** | **2.5** |
| 大皮卡（如 Mercedes X-Class）| **1** | **最軟** |
| 直線加速 | **65** | **1** |

原則：**前軟後硬** → 改善推頭；**前硬後軟** → 改善甩尾。**前防傾桿尤其重要**。

## 症狀處理（依手感調整）

| 症狀 | 處理方式 |
|------|---------|
| **轉向過度**（太能拐、後輪甩出去）| **前硬一點、後軟一點**，往中間靠 |
| **轉向不足**（推頭、拐不過去）| 先套「前 1 / 後 65」；若仍推頭 → 改彈簧（見 [彈簧與車高](./彈簧與車高.md)）|
| 操控過於靈活 | 前側拉硬、後側拉軟，**但整體仍保持前軟後硬** |

## 禁忌與原則

> ⚠️ **不要「前硬後軟」**。

- 保持 **前軟後硬**
- 讓前後比值儘量接近（約 **70 ~ 75 %**），**不要讓兩者交叉**

## 注意

**性能越強，防傾桿影響越大**，需反覆依試車表現調整。

## 「165 ARB」社群梗（英文圈）

HokiHoshi 2025 影片提到：

> 「**前 ARB `1`，後 ARB `65`**」幾乎成了社群梗，因為太多人這樣設。
> 這不是壞建議——**特別是 AWD 車很受惠**。

這正好對應上方表格中**多家中文來源一致**的「四驅／公路／拉力越野車：前 1 / 後 65」設定。HokiHoshi 替這個共識設定取了「**165 梗**」這個暱稱，方便英文社群討論。

### HokiHoshi 的補充原則

- 若**過度轉向太嚴重** → **提高前 ARB**
- **RWD 與 FWD 車**通常**前後都頂到最大**也很好（與中文硬核版的「後驅 10/5.6」是不同流派）
  - **RWD 想要更多 rotation/oversteer** → 降前 ARB
  - 想要**更穩定/understeer** → 降後 ARB
