# 過彎三段診斷與症狀對策

> **🩺 用比喻理解**：診斷車況像**看醫生**——你不能只說「車怪怪的」，要明確指出：
> - 是**進診間時不舒服**（入彎）？
> - **坐下後**不舒服（中段）？
> - 還是**起身離開時**不舒服（出彎）？
>
> 不同段對應不同「處方」（調校項目）。**指錯段就吃錯藥**——車不會變好。

> 把過彎拆成三段——**入彎、中段、出彎**——分別診斷症狀並對症下藥，是 HokiHoshi 系列教學的核心方法論。
>
> 一輛車在三段中可能**同時出現**不同症狀，**段與段之間的轉換**也要分開看。診斷時要明確指出**哪一段出問題**，再從對應的對策清單中挑選調整。

---

## 三段定義

| 段 | 駕駛動作 | 物理重點 |
|---|---|---|
| **入彎（Entry）** | 鬆油門 → 踩煞車 → 開始打方向盤 | 重心前移、煞車穩定性、轉向反應 |
| **中段（Mid Corner）** | 已穩定在彎道中、油門接近平衡 | 側向抓地、車身傾倒、外側輪負荷 |
| **出彎（Exit）** | 開始重新加油 → 拉直方向盤 | 動力下放、後輪抓地、扭矩分配 |

> 良好設定的目標：**入彎能切進去、中段能維持傾倒、出彎能穩定加速**。

---

## 三驅動的常見症狀傾向

| 驅動 | 容易在哪段出問題 |
|---|---|
| **AWD（四驅）** | 入彎、中段 understeer 為主 |
| **FWD（前驅）** | 入彎、中段 understeer 為主 |
| **RWD（後驅）** | 出彎 oversteer 為主（重新踩油時後輪甩出） |

---

## 入彎 Understeer（轉向不入）

> **症狀**：車像「不想轉」、抗拒方向輸入；煞車中車頭往外推。

> ⚠️ **重要警告**：**煞車中 understeer 是正常的**——煞車時重心前移、後輪抓地降低，本來就會 understeer。**不要過度調整**。

### 對策清單

從便宜到激進依序試：

1. **前胎壓**：調整以提升峰值抓地（見 [胎壓](./胎壓.md)）
2. **彈簧 / ARB**：軟前彈簧或軟前 ARB；**或**硬後彈簧或硬後 ARB
3. **束角**：加一點前 toe out（≤ 0.3°）或一點後 toe out
4. **下壓力**：加前下壓力 / 減後下壓力
5. **前 diff decel**：降低前差速器減速鎖定
6. **阻尼（煞車中專屬）**：加前 bump 或減前 rebound——若主要在**煞車中** understeer
7. **煞車平衡**：滑桿往前推（一般應偏前；具體效果請對照 [煞車調校](./煞車調校.md)）

> 配合 [遙測使用指南](./遙測使用指南.md) 觀察懸吊與胎溫，能更精確找出原因。

---

## 中段 Understeer

> **症狀**：車入彎好，但**中段又「拉直」**——明明應該維持傾倒到出彎，卻被拉回直線。

> 一個良好設定**入彎傾倒後應維持到出彎**。

### 對策清單

1. **ARB**：**硬後**而非軟前——重點放在後 ARB
2. **後懸吊**：硬後彈簧
3. **下壓力**：檢查設定，視情況調整
4. **後阻尼**：加後 rebound 與後 bump
5. **前胎壓 / camber / caster**：玩這三個找前胎峰值抓地

---

## 出彎 Understeer

> **症狀**：開始重新踩油時車不肯轉、推頭出去。

> 主要是**差速器設定**問題。

### 對策清單（差速器為主）

1. **中央 diff（AWD）**：偏更後（見 [差速器](./差速器.md)）
2. **後 diff accel**：加後差速器加速鎖定
3. **前 diff accel**：減前差速器加速鎖定（避免 plow understeer）

---

## Oversteer（三段通用）

> **症狀**：車尾甩出去、轉得太多、容易打轉。
>
> **核心對策**：**把上述三段 understeer 對策全部反向操作**——什麼導致一個方向的 understeer，反過來就會 oversteer。

### RWD 出彎 Oversteer 的特殊處理

RWD 出彎甩尾常見，通常**先試改善油門控制**而非調車：
- **節流量管理**比設定更重要
- 真要調車：減後 diff accel、增大後 ARB、提升後輪抓地

### RWD 抓地 build 的重點

> 增加後抓地但**不要引入太多 understeer**。重點放在：
> - **下壓力**
> - **bump 阻尼**
> - **定位**（camber/toe）

---

## 其他常見症狀對策

| 症狀 | 對策 |
|---|---|
| **快速 left → right 過渡** under/oversteer | 阻尼與 ARB（見 [阻尼](./阻尼.md) [防傾桿](./防傾桿.md)） |
| 過 curb（縁石）容易把車甩飛 | **降低 bump 阻尼** |
| **胎不夠熱 + understeer** | 全車加 **toe out**（一次解決兩個問題） |
| **胎不夠熱 + oversteer** | 全車加 **toe in** |
| 胎不夠熱（無方向偏好） | 加下壓力 |
| **車太彈、重心轉移過大** | 加 bump 阻尼 + 加彈簧硬度 |
| **煞車不夠強** | 多管齊下：減前 camber 但加 caster 補回；煞車平衡往後（依[煞車](./煞車調校.md)說明）；硬前懸吊把更多煞車抓地分到後輪（**僅在前煞已最大化時有效**） |
| **極速不夠** | 降下壓力 + 調整 [final drive](./齒比.md) |

## 工作流程建議

### 1. 先用 base tune（起手值）跑

從 [三車種公式速查](./三車種公式速查.md) 或 [QuickTune 套用流程](./QuickTune套用流程.md) 找到對應車型的起手值套上。

### 2. 在固定賽道反覆跑相同彎道

推薦在 **Festival Sites（節慶站）的賽道**或自己常跑的圈：
- 同一個彎反覆跑，觀察車的反應
- 開啟 [遙測](./遙測使用指南.md)（PC: `T` 鍵；Xbox: 設定 ANNA → 方向鍵下）

### 3. 明確指出哪段出問題

不要只說「車很怪」——**指出具體段**：
- 入彎時轉不過去 → 入彎 understeer
- 中段車身被拉直 → 中段 understeer
- 出彎踩油車就甩 → 出彎 oversteer
- 等等

### 4. 從對應對策清單**從便宜到激進**逐項試

每次只改一項或一組相關項，跑同一個彎驗證。

### 5. 別怕反覆

> 「真正的學習在做中」——HokiHoshi 強調每輛車不同、每位駕駛不同，永遠有例外。

---

## 實證效果（HokiHoshi R35 對照範例）

> HokiHoshi 在 S1 R35 build 上對比三種調校在同一賽道（**故意保留前 aero 鎖定**以放大 understeer）：

| 調校狀態 | 結果 |
|---|---|
| **原廠調校** | 嚴重 understeer，缺前 aero 補不回，難開、不爽 |
| **Base tune 套用後** | 圈速明顯改善——base tune 已預先補償 AWD 預期的 understeer |
| **完整精調 tune** | 大幅領先原廠，**完全沒升級任何零件、PI 不變、性能完全是調校榨出來的** |

> **這就是調校的價值：不增加 PI 也能榨出大量性能。**
