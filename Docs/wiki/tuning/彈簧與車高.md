# 彈簧與車身高度

> **🛏️ 用比喻理解**：
>
> **彈簧硬度** = **床墊軟硬度**：
> - **軟床（軟彈簧）**：舒服、吸震好、**但翻身會晃**（重心轉移大）
> - **硬床（硬彈簧）**：支撐強、**反應快**、但路面顛簸感受強，太硬會**失去抓地**
>
> **車高** = **床的高度**：
> - **越低**：重心低、不易翻車、極速高——**但容易磨地（bottom out）**
> - **越高**：穩定但易翻；**FH5 物理特性**：抬高反而**過彎抓地更好**（反直覺！）
>
> **前後比例**：像**長板凳一頭軟一頭硬**——前軟後硬促轉向、前硬後軟促穩定。

> **修訂 2026-04-21**：從《萌新攻略第九期》補入**公路車前後比例 0.75 : 1**、**推頭時的彈簧口訣**（低速彎軟／高速彎硬）、**車高對照表**（公路漂移最低／拉力越野最高）與 **Apollo 前 8 / 後 11 的特例處理**。

可調前後側**彈簧硬度**與**車身高度**。

## 彈簧硬度

### 軟硬特性

- **軟** → 易壓縮、重心易轉移、濾震好
- **硬** → 不易壓縮、重心不易轉移、濾震差

不同懸掛／不同車的可調範圍不同，若不夠只能回配件處換懸掛。

### 軟硬推薦

越野車、拉力車彈簧通常比賽車懸掛軟：越野需在凹凸路面保持穩定；賽車需清晰感知路面；拉力車介於兩者之間。

| 車種 | 建議設定 |
|------|----------|
| **公路車**（範圍做法） | 先設前後 **15** 或 **150**（依車型），讓加速時重心轉移少、減速時轉向靈活。前側若調不到就拉到最小值 |
| **公路車**（比例公式） | **前後比例 0.75 : 1**。實際套車時再**稍微硬一點**，例如 **140 / 100**（約 1.4 / 1.0 的感覺）|
| 拉力車（範圍做法） | 前 **65** / 後 **90** |
| 拉力 / 越野車（公式做法） | **前後軟硬相同**，取兩側最小值中的較大值<br>範例：前最軟 50、後最軟 56 → 前後均設為 **56** |
| 越野車（範圍做法） | 彈簧**先拉到最低**，再統一前後調整為最小值較高的那個數值 |

以上數值非絕對，需依試車表現調整。

> **特別提醒**：越野、拉力車彈簧過軟或過硬，飛坡後可能姿態過激；最理想狀態是**起飛後車身維持水平，平穩落地**。

### 推頭時的彈簧處理（核心口訣）

| 狀況 | 動作 |
|------|------|
| **低速彎**拐不過去（推頭）| **整體拉軟** |
| **高速彎**拐不過去 | **整體拉硬** |
| 高速彎太靈敏、容易抖 / 失控 | 也往「整體拉硬」方向（但不要太硬）|

**通用口訣**：
- **越軟 → 低速彎越好**
- **越硬 → 高速彎越好**

#### 為何常優先改硬？

- 高速彎優化後，**低速彎的推頭可以靠鬆油門 / 輕給油來緩解**。
- 推頭的根本原因是「車還有向前的慣性」；沒動力時轉向自然靈敏。
- 所以常見做法：**先硬 → 低速彎用油門控制**。

### 硬核版：依驅動方式細分

| 驅動方式 | 前彈簧 | 後彈簧 |
|---------|--------|--------|
| **前驅 / 四驅** | 最軟 | 約在 **2/3** 位置 |
| **四驅（另一思路）** | 最軟 | 最硬 |
| **後驅** | 不超過調整範圍的 1/3，**初始約 80** | **初始約 100**（依試車調整）|
| **大馬力後驅** | 前後可**一樣**，甚至都用最軟 | 同左 |
| **拉力懸掛（共通）** | 不超過行程的 **1/6** | 不超過 **1/4** |
| **拉力調校** | 偏軟 | 偏硬（前軟後硬提升轉向）|
| **越野調校** | 前後軟硬度**一樣**（起跳時不推頭不翹頭）| 同左 |
| **輕量越野車** | 前後都最軟 | 同左 |
| **直線加速** | 最硬 | 最軟 |

> ⚠️ 遊戲早期流傳的「**四代彈簧計算公式**」計算出來的結果有**嚴重轉向不足**，**不推薦使用**。

> **英文社群旁證**：HokiHoshi 2021 影片同樣指出「過去（FH4 時代）我用一套公式設基準值，**但 FH5 懸吊物理大幅改善，那套公式不再那麼好用**」——中英文社群在此判斷一致。

原則補充：
- **前軟後硬** → 改善轉向不足
- **前硬後軟** → 減少轉向過度、增加轉向不足
- **彈簧越軟** → 重心轉移越快
- **彈簧越硬** → 操控響應越好，但太硬容易失去抓地力

---

### HokiHoshi 2025：FH5 meta 偏軟趨勢

> 因 FH5 給的抓地多，**跑得快的 build 普遍很軟**——
> 甚至設**到最小值**，或**前比後更軟**以誘發更多 rotation。

> 此觀點與既有「軟硬推薦」表中部分數值（如越野輕量車前後都最軟、大馬力後驅前後都最軟）方向一致，但 HokiHoshi 把這個趨勢推廣到**所有 meta build**，而非只限特定車種。

## 車身高度

### 特性

- 一般越低 → 越穩、速度略快
- 越野、拉力車車身越低 → 彈簧行程越短、減震差、**涉水減速更多**
- 越野、拉力車為減少涉水減速會調高，達 **16 cm** 時涉水阻力很小
- 但懸掛越高 → 穩定性越差、越易打滑

### 推薦高度

| 車種 | 車身高度 |
|------|----------|
| 公路賽 / 漂移 | **最低** |
| 越野車 | **不低於 15 cm**（第九期：直接「最高」）|
| 拉力車 | 前後 **13.6 cm**（第九期：直接「最高」）|
| 公路車 | 盡可能最低，建議前後一致；部分車型可設「前高後低」以提升極速與加速 |

### ⚠️ HokiHoshi 2025：抬高車高反而改善過彎抓地（Forza 物理特性）

> **這是 FH5 物理引擎的反直覺特性**：抬高車高有時**反而改善過彎抓地**——即使是純公路車。

| 派別 | 立場 |
|------|------|
| 中文社群（多家） | **公路車盡可能降低**（重心低 → 操控好、空力好）|
| HokiHoshi 觀察 | **抬高車高可改善過彎抓地**，但**犧牲一些穩定性**；HokiHoshi 個人不喜歡那種視覺與感覺，仍降低公路車 |
| **拉力 / 越野** | 兩派一致：**頂到最大** |

> 兩派不衝突——HokiHoshi 提供**另一條取捨路徑**：若你的公路 build 過彎抓地不足，可實驗抬高車高換取抓地，犧牲一點穩定。

> **觀念底層**：FH5 的下壓力／空力與真實世界不同（見 [基礎概念](./基礎概念.md) FH5 物理特性章節），這也是為何抬高車高在 FH5 中不見得是壞事。

### 前後懸殊的特例

有些車最低高度**前後懸殊**，例如 **Apollo** 前 **8 cm** / 後 **11 cm**：

- 若車感不穩 → 把前側**拉高一點**，儘量讓前後等高，至少保證穩定
- 懶得調 → 就照上表二擇一：公路／漂移最低；拉力／越野最高

### 硬核版：依類型細分

| 調校類型 | 設定 |
|---------|------|
| **公路**（一般）| 用**預設數值**，以「高的一側」配平 |
| **公路特例**（KSG CCGT、法拉利 FXX 等）| 可**拉高車頭**，提高操控可控性，甚至提升極速 |
| **拉力 / 越野** | 前後都**最高** |
| **直線加速** | 前輪**最高**、後輪**最低** |
| **漂移** | 前後**統一最低** |
