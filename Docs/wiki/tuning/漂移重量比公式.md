# 漂移車重量比套用公式

漂移車的**防傾桿 / 彈簧 / 阻尼**較特殊，不是固定值，而是依**車輛前後重量比**推算。

## 公式

```
(最大值 − 最小值) × 重量比 + 最小值
```

### 步驟

1. **進入選車界面**查看車輛前後重量比（例：前 48 % / 後 52 %）
2. 依上方公式計算
3. 套用規則：
   - 要改「**前側**」的數值 → 用**前側**重量比
   - 要改「**後側**」的數值 → 用**後側**重量比

## 套用範例

假設某車前重量比 **48 %**，某滑桿最小 **10**、最大 **60**：

- 前側數值 = `(60 − 10) × 0.48 + 10` = **34**
- 後側數值 = `(60 − 10) × 0.52 + 10` = **36**

同一條公式套給**防傾桿、彈簧、阻尼**的前後兩側滑桿。
