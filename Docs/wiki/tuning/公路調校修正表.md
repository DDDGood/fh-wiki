# 公路調校修正表

本表在 [QuickTune 基準值總表](./QuickTune基準值總表.md) 的基礎上，依公路情境疊加偏移量。

## 核心觸發條件

- **馬力 < 400 hp 且使用原廠胎或無可調懸吊** → 直接用基準值，不套用本表。
- **馬力 ≥ 400 hp 且使用可調懸吊 + 升級胎種** → 套用本表偏移。
