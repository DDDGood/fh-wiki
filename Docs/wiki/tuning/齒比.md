# 齒輪（傳動比）調校

> **修訂 2026-04-21**：從《萌新攻略第九期》補入**固定比值公式**（6 檔 / 7 檔 / 8 檔以上）與**終減速比調整法**（拉到最高再往回撥 0.1 的方法）。

## 界面說明

- **最上方**：最終傳動比
- **下方**：各檔位傳動比
- 數值越往右 → 右下角轉速曲線斜率越大

### 圖表解讀

| 元素 | 含義 |
|------|------|
| 橫軸 | 車速（最右代表理論極速，不同車不同） |
| 縱軸 | 發動機轉速 |
| 斜線 | 該檔從最低到最高轉速可達的速度區間 |
| 紅線 | 最高轉速 |

斜率越大 → 加速越快，但該檔理論極速越低。

## 調校方法（通用）

- 關注左下角的「**0-160 km/h 加速**」與「**極速**」。
- 配合選配件時記下的功率曲線，讓每個檔位的最低與最高轉速段都落在**引擎馬力、扭矩最佳工作範圍**。
- 每升一檔，引擎有效功率會降低一點。

### 車型取向

| 車型取向 | 檔位建議 |
|----------|----------|
| 換檔慢、長直線加速車 | 檔位不要太密 |
| 操控車、複雜路段車 | 檔位密一點，過彎更穩、更好控制車速 |

### 通用微調做法

1. 調好一檔後，下一檔轉速盡量比前一檔**高約 300 轉**；越往後檔位之間的差距越小。
2. 最終檔的轉速最低點在各檔連成的線上應**呈弧形**。
3. 最後用**直線加速測試**微調。

---

## 固定比值公式（公路車起手套用）

以 **6.2 L V8 引擎**為範例，作者提供的「一定好用」的固定比值。末位三位小數都補 `000`（如 `3.000 / 2.000 / 1.500`...）。

| 檔位數 | 各檔比值 |
|--------|----------|
| **6 檔** | **3.00 / 2.00 / 1.50 / 1.20 / 1.00 / 0.85** |
| **7 檔** | **3.70 / 2.35 / 1.65 / 1.30 / 1.10 / 0.95 / 0.85** |
| **8 檔以上** | 依 7 檔設置，**第 8、9、10 檔均為 0.85** |

---

## 終減速比調整法（關鍵步驟）

套用上述各檔比值後，用以下流程調整**終減速比（Final Drive）**：

1. **先輸入上述檔位數值**
2. **把終減速比的滑桿拉到最高（最左）**
3. 觀察此時**理論極速**（例：325.1 km/h）
4. 將滑桿往右拉到「**350 刻度前一格**」的位置，記錄極速變化（例：320.3 km/h）
5. 嘗試繼續往右拉一半 → 若極速**突然大幅下掉**（例：到 317.5），就**不要再往右**
6. 往回拉到「那半格的一半再往前一點」
7. 接下來**每次往左撥 0.1**，觀察極速增加幅度：
   - 例：從 317.5 → 318.4（+0.9）→ 318.9（+0.5）→ 再往左 +0.3
   - **當增量小於 0.5 時就停下**，這就是該車的極速極限
8. 最終落點（範例車）：**終減速比約 3.5**

### 依用途微調

| 用途 | 終減速比方向 |
|------|-------------|
| 跑圈圖（拉力居多，用不到那麼高的極速）| 可**往右（加速方向）拉一點點** |
| 跑公路比賽、要極速 | 按「小於 0.5 就停」的方法調到頂 |
| 漂移車 | **偏加速** |

> 不用追求理論極速，能達到「改到最大的那個極速」就夠了。

## 功率帶（Power Band）原理

改裝頁面右側的 **Engine Dyno Graph**：
- **紅線** = 扭矩（決定加速度）
- **黃線** = 馬力 / 功率（決定速度）
- **橫軸** = 發動機轉速
- **縱軸** = 發動機輸出功率

**峰值扭矩 → 峰值馬力**之間的轉速區間，就是該發動機的**理想工作轉速（功率帶）**。

例：某引擎理想轉速大約 **5000–9300 rpm**。

### 齒輪比調校目標

- **從二檔開始，每一檔最高轉速都落在理想功率帶內；每檔起始轉速依序遞增**
- 調整**最終傳動比**，讓車輛在**最後一檔**拉到功率 / 扭矩峰值時剛好達到極速——最後一檔末端**頂到右上角**
- 理論極速只有在**無額外下壓力**、拉力胎與理想最終傳動比下才能達到

---

## 十速變速箱的處理

**選十速變速箱常是為了配重 / 卡 PI，不代表要用十個檔**——賽車中除非萬轉引擎，否則很難用到 8 檔以上。

此時透過最終傳動比，把十速當 **6 檔或 7 檔**用。

### 起步 / 出彎加速打滑時

**把低檔拉長**：

| 變速箱類型 | 處理方式 |
|-----------|---------|
| 賽車變速箱 | 把**第一檔**的傳動比調長 |
| 跑車變速箱 | 把**最終傳動比**調長 |

> UP 主另分享了幾個「萬能傳動比」，可直接套用，再依上述邏輯調整最終傳動比。
