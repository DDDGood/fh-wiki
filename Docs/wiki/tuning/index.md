# 🔧 車輛調校

> **比喻**：車像一套**量身訂做的西裝**——買來時尺寸大致合身（原廠調校），但要**修腰、收袖、改褲管**才真正貼合你的體型（駕駛風格）與場合（賽道）。**調校就是這個量身修改的過程**——不增加 PI，但讓車**真正合手**。

懸吊、差速器、阻尼、齒比、胎壓等可調參數的原理與實用數值。

---

## 🎮 9 大調校項目（依遊戲面板順序）

> 遊戲調校面板由左到右順序，每個項目附**比喻**＋**控制什麼**：

### 1️⃣ [胎壓](./胎壓.md) — Tire Pressure
🎈 **像氣球**：太硬反彈快但接觸面小、太軟黏地但易發熱
**控制**：輪胎軟硬度與接觸面 → **抓地力與升溫速度**
**單位**：本檔以 **PSI** 為主（FH5 預設顯示）

### 2️⃣ [齒比](./齒比.md) — Gearing
🚲 **像腳踏車變速**：短齒比起步快但極速低、長齒比反之
**控制**：引擎轉速與車速的轉換比 → **加速 vs 極速**
**重點**：Final Drive 是最重要的設定

### 3️⃣ [四輪定位](./四輪定位.md) — Alignment
👟 **像穿高跟鞋**：適度傾斜讓鞋底「壓滿地」（過彎時整個胎面接地）
**控制**：Camber（外傾）／Toe（束角）／Caster（後傾）
**通則**：Caster 直接拉到 **7°**

### 4️⃣ [防傾桿](./防傾桿.md) — Anti-Roll Bars
🪢 **像連接左右輪的橡皮筋**：一邊壓下去另一邊跟著動 → 整車側向穩定
**控制**：左右側向耦合度 → **過彎側傾與靈動感**
**社群梗**：「**165 ARB**」（前 1 / 後 65）

### 5️⃣ [彈簧與車高](./彈簧與車高.md) — Springs
🛏️ **像床墊軟硬度＋床的高度**：軟床舒服但晃、硬床支撐強但顛
**控制**：彈簧硬度＋離地間隙 → **重心轉移與抓地**
**FH5 反直覺**：抬高車高有時**反而提升過彎抓地**

### 6️⃣ [阻尼](./阻尼.md) — Damping
🧽 **像床墊海綿吸收力**：控制彈簧晃多少（回彈太快蹦三尺、太慢車輪卡身內）
**控制**：Rebound（回彈）＋Bump（衝擊）→ **濾震與穩定性**

### 7️⃣ [下壓力](./下壓力.md) — Aero
✈️ **像戰鬥機翅膀反過來裝**：速度越快壓得越緊
**控制**：前 / 後下壓力 → **過彎抓地** vs **極速**（drag 代價）

### 8️⃣ [煞車調校](./煞車調校.md) — Brake
🚲 **像踩腳踏車前後煞車的比例**
**控制**：制動力分布＋煞車壓力 → **煞車穩定性**

### 9️⃣ [差速器](./差速器.md) — Differential ⭐ 「車的靈魂」
🚶 **像兩條腿走路時的協調**：左右腳可以一快一慢，過彎才不會被內輪卡住
**控制**：前 / 後 / 中央差速器的鎖定百分比 → **動力如何送到地面**

## 🩺 快查：症狀 → 該調哪裡

### 過彎症狀

| 症狀 | 主要對策 | 對應檔 |
|------|---------|--------|
| **車不肯轉、推頭**（understeer）| 軟前彈簧／軟前 ARB／加前下壓力／前 toe out | [防傾桿](./防傾桿.md) [彈簧](./彈簧與車高.md) [下壓力](./下壓力.md) [四輪定位](./四輪定位.md) |
| **車尾甩出、打轉**（oversteer）| 硬前 ARB／硬前彈簧／降後 diff accel／加後下壓力 | [防傾桿](./防傾桿.md) [差速器](./差速器.md) |
| **入彎不轉、出彎甩** | 三段都有問題——查 [三段彎道診斷](./三段彎道診斷.md) | [三段彎道診斷](./三段彎道診斷.md) |
| **入彎時推頭** | 加前 bump 阻尼／降前 diff decel | [阻尼](./阻尼.md) [差速器](./差速器.md) |
| **中段彎被拉直** | 硬後 ARB／硬後彈簧／加後阻尼 | [防傾桿](./防傾桿.md) [阻尼](./阻尼.md) |
| **出彎踩油就甩** | 加後 diff accel／加後抓地（下壓力、bump、camber）| [差速器](./差速器.md) [下壓力](./下壓力.md) |
| **過 curb 容易甩飛** | 降 bump 阻尼 | [阻尼](./阻尼.md) |
| **快速 left→right 切換 over/understeer** | 阻尼與 ARB | [阻尼](./阻尼.md) [防傾桿](./防傾桿.md) |

### 速度症狀

| 症狀 | 主要對策 | 對應檔 |
|------|---------|--------|
| **極速不夠** | 降下壓力、調 final drive | [下壓力](./下壓力.md) [齒比](./齒比.md) |
| **加速不夠** | 縮短齒比、看是否要加馬力 | [齒比](./齒比.md) |
| **常卡在檔位之間** | 微調個別檔位齒比 | [齒比](./齒比.md) |
| **加馬力後末檔頂不到** | 把 final drive 拉長 | [齒比](./齒比.md) |

### 抓地症狀

| 症狀 | 主要對策 | 對應檔 |
|------|---------|--------|
| **胎太冷（藍）** | 降胎壓／加 toe／加下壓力 | [胎壓](./胎壓.md) [四輪定位](./四輪定位.md) |
| **胎太熱（橘紅）** | 加胎壓／降 camber | [胎壓](./胎壓.md) [四輪定位](./四輪定位.md) |
| **胎內外溫差太大** | 減負 camber（>20°F 差）| [四輪定位](./四輪定位.md) |
| **胎中間最熱** | **胎壓太高** | [胎壓](./胎壓.md) |
| **胎內外都熱、中間冷** | **胎壓太低** | [胎壓](./胎壓.md) |
| **車太彈、重心晃** | 加 bump 阻尼＋加彈簧硬度 | [阻尼](./阻尼.md) [彈簧](./彈簧與車高.md) |

### 煞車症狀

| 症狀 | 主要對策 | 對應檔 |
|------|---------|--------|
| **煞車距離太長** | 提高煞車壓力、檢查 ABS | [煞車調校](./煞車調校.md) |
| **煞車中車尾甩** | 平衡偏前 | [煞車調校](./煞車調校.md) |
| **煞車中推頭** | 平衡偏後（注意 FH5 滑桿） | [煞車調校](./煞車調校.md) |
| **煞車鎖死打滑** | 降煞車壓力、開 ABS | [煞車調校](./煞車調校.md) |

> 用 [遙測使用指南](./遙測使用指南.md) **量化驗證**症狀（懸掛壓縮、胎溫色區、抓地力綠圈）。

## 🚗 各車種的調校重點

### 公路車（Road / Track）
**重點**：[胎壓](./胎壓.md) → [差速器](./差速器.md) → [防傾桿](./防傾桿.md) → [下壓力](./下壓力.md)
- 胎質：原廠 / Sport / **Rally / Drift**（HokiHoshi 2025 觀點）
- 車高：盡可能**最低**
- ARB：常見「**165 梗**」（前 1 / 後 65）
- Caster：頂到 **7°**

### 拉力車（Rally）
**重點**：[彈簧](./彈簧與車高.md) → [阻尼](./阻尼.md) → [胎壓](./胎壓.md)
- 胎質：**Off-road race**
- 車高：**頂到最大**（涉水減速）
- 胎壓：**14.5 PSI / 1.0 bar**（低壓早暖、土路飄著走）
- Bump 阻尼：偏軟（吸震）

### 越野車（Cross-Country）
**重點**：[彈簧](./彈簧與車高.md) → [阻尼](./阻尼.md) → [差速器](./差速器.md)
- 胎質：**Off-road race**
- 彈簧：軟（飛坡水平落地為佳）
- 差速器：**100 / 100**（避免懸空輪空轉）
- 防傾桿：**前後接近**（起跳不推頭不翹頭）

### 漂移車（Drift）
**重點**：[差速器](./差速器.md) → [防傾桿](./防傾桿.md) → [胎壓](./胎壓.md) → [漂移重量比公式](./漂移重量比公式.md)
- 後 diff：**100 / 90**（鎖越死越好飄）
- 漂移專用算法：依重量比套公式（[漂移重量比公式](./漂移重量比公式.md)）
- Camber：**負大**（建議拉滿）
- 後傾角：**頂滿**

### 直線加速（Drag）
**重點**：[齒比](./齒比.md) → [彈簧](./彈簧與車高.md) → [差速器](./差速器.md)
- 彈簧：**前最硬 / 後最軟**
- 阻尼：**前 20 / 後 1**
- ARB：**前 65 / 後 1**
- 車高：**前最高 / 後最低**
- 不要升空力套件

## 🛤️ 閱讀順序建議

**新手想直接套用**：
1. [基礎概念](./基礎概念.md) 5 分鐘理解傳動／引擎布局
2. [三車種公式速查](./三車種公式速查.md) 直接填數值
3. 跑一圈 → 用上方「症狀 → 對策」表微調

**想理解原理**：
1. [基礎概念](./基礎概念.md)
2. 依遊戲面板順序：1 胎壓 → 2 齒比 → 3 四輪定位 → 4 防傾桿 → 5 彈簧 → 6 阻尼 → 7 下壓力 → 8 煞車 → 9 差速器
3. 漂移車另讀 [漂移重量比公式](./漂移重量比公式.md)

**車跑不順時**：
1. 用 [遙測使用指南](./遙測使用指南.md) 找出懸吊／胎溫／camber 是否異常
2. 用 [三段彎道診斷](./三段彎道診斷.md) 明確指出哪段出問題並對症調整

**進階追求精細**：
1. [QuickTune 套用流程](./QuickTune套用流程.md) 入手
2. 依 Part 1~7 逐步查表疊加
