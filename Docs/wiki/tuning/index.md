## 閱讀順序建議

**新手想直接套用**：看 [三車種公式速查](./三車種公式速查.md)，按表填入數值即可。

**想理解原理**：
1. 先讀 [基礎概念](./基礎概念.md) 理解傳動與重量
2. 再讀 [胎壓](./胎壓.md) 與 [差速器](./差速器.md) 打底
3. 懸吊進階依 [四輪定位](./四輪定位.md) → [防傾桿](./防傾桿.md) → [彈簧與車高](./彈簧與車高.md) → [阻尼](./阻尼.md) 的順序
4. 漂移車另讀 [漂移重量比公式](./漂移重量比公式.md)

**進階玩家追求精細**：從 [QuickTune 套用流程](./QuickTune套用流程.md) 入手，依 Part 1~7 逐步查表疊加。
