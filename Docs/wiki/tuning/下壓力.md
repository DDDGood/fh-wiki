# 下壓力（空氣動力）調校

> **修訂 2026-04-21**：從《萌新攻略第九期》補入**極速範例**（某車不加空力 319.3 km/h → 拉滿 308 km/h）、**漂移車前後明確值**（前最右／後最左）與「**不為壓 PI 分就別裝空力**」觀點。

有些車原廠解鎖，有些需選配。調整時關注左邊 **側向 G 力** 與 **極速**。

## 基本原則

- **向右調** → 下壓力大、極速降低
- **向左調** → 下壓力小、極速高

### 依車輛類型取向

| 車輛類型 | 策略 |
|---------|------|
| **操控優先的車** | 下壓力**拉滿** |
| **操控過剩 / 速度優先的車** | 適當調低，換更大極速 |
| **不需要壓 PI 分** | **完全不裝空力** |

### 前 / 後取捨（公路 / 拉力 / 越野）

- **前側**解鎖下壓力基本**拉滿**
- **後側**適當取捨，避免過度影響極速

### 漂移車特殊設定

- **前空力**：拉到**最右**（最大）
- **後空力**：拉到**最左**（最小）

## 範例

某車實際比對：

| 設定 | 極速 |
|------|------|
| 不加空力 | **319.3 km/h** |
| 空力拉到最高 | **308 km/h** |

→ 找一個「你能接受的操控 / 極速平衡點」即可。

## 觀念提醒

> **不是為了壓分就別裝空力**，很多車純粹為了留 PI 額度去裝空力，沒其他理由。

## 特別注意

**拉力車 / 越野車**：後側下壓力過大，飛坡後易**瘋狂翹頭**，要特別注意。

## 硬核版：依驅動方式與 PI 階級細分

| 驅動方式 | 前下壓力 | 後下壓力 |
|---------|---------|---------|
| **前驅 / 四驅** | **拉到最高** | 不超過**一半**位置 |
| **後驅**（A 級以下） | **拉滿** | 先視為**最低**，再依試車逐步調大 |
| **後驅**（S1 以上大馬力） | 拉滿 | 依需求偏大，甚至**拉滿** |
| **直線加速** | — | **不需要**升級空力套件 |

### 原則

- 後下壓力越大 → 後輪抓地力越強，但太大會造成**轉向不足**
- **PI 等級越高的車 → 後下壓力需求越大**
- 初始設定都需要在**試車過程中依反饋調整**
