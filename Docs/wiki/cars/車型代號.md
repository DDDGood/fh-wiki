# 車型代號（Chassis Codes）

> **🆔 用比喻理解**：Chassis code 像車的「**世代身分證**」——
>
> 你不會說「我朋友是台灣人」就讓人知道是誰；你會說「他是 1995 年出生、住台北」。**車也一樣**——「Mazda RX-7」太籠統，「**FD**」立刻指向 1992-2002 那台跑車界傳奇。
>
> 同一車名（如 RX-7、Skyline、911）會跨幾十年改了好幾代，**chassis code 是分辨世代的快速密碼**。

> **同一車系的不同世代／版本**——以**字母 + 數字**形式區分。看到隨機字母+數字組合形容車時，**多半是 chassis code**。

---

## 為什麼需要 Chassis Code

同一車名（例如 Mazda RX-7）有多個世代，性能差異極大。如果只說「RX-7」，無法分辨指的是 1980 年的初代還是 1990 年代的末代。Chassis code **快速精確**指明世代。

---

## 常見 Chassis Codes

### 日系車

#### Nissan S 底盤（Silvia / 240SX 系列）

| 代號 | 對應 |
|------|------|
| **S13** | 240SX / Silvia 第三代（1988-1994） |
| **S14** | 240SX / Silvia 第四代（1994-1999） |
| **S15** | Silvia 第五代（1999-2002，日本限定） |

> 漂移文化中極受歡迎的車系——每代各有偏好群體。

#### Mazda RX-7

| 代號 | 對應 |
|------|------|
| **FB**（也叫 SA22C）| 一代 RX-7（1978-1985） |
| **FC** | 二代 RX-7（1986-1991） |
| **FD** | 三代 RX-7（1992-2002，最終代）|

#### Mazda MX-5 Miata

| 代號 | 對應 |
|------|------|
| **NA** | 一代（1989-1997，圓燈經典） |
| **NB** | 二代（1998-2005） |
| **NC** | 三代（2005-2015） |
| **ND** | 四代（2015-至今，目前最新） |

> ⚠️ **NA Miata 的「NA」**指 chassis code，**不是 Naturally Aspirated**！同字面不同含義。

#### 其他常見日系

- **AE86** — Toyota Corolla GT-S（漂移聖物）
- **R32 / R33 / R34 / R35** — Nissan Skyline GT-R 各世代
- **JZA80** — Toyota Supra 四代（搭載 2JZ 引擎那台）

### 美系車

#### Chevrolet Corvette

| 代號 | 對應 |
|------|------|
| **C1** ~ **C8** | 各世代（C1 = 1953 一代；C8 = 2020 中置引擎當代）|

> Forza 中常見 C5、C6、C7、C8 同名「Corvette」但用 chassis code 才能精確分辨。

### 歐系車

#### BMW 3 系

| 代號 | 對應 |
|------|------|
| **E30** | 1982-1994（M3 經典） |
| **E36** | 1990-2000 |
| **E46** | 1998-2006 |
| **E90** | 2005-2013 |
| **F30** / **G20** | 後續世代 |

#### Porsche 911

| 代號 | 對應 |
|------|------|
| **930** | 1975-1989（早期 turbo 版） |
| **964** | 1989-1994 |
| **993** | 1993-1998（最後一代氣冷）|
| **996** / **997** / **991** / **992** | 後續水冷世代 |

## 在 Forza 中辨識

> Forza 不一定在車名上印 chassis code——但社群討論一定會提。

範例：
- 「**FC vs FD**」 = 二代 RX-7 vs 三代 RX-7
- 「**E46 M3**」 = 1998-2006 那代的 BMW M3
- 「**S15 Silvia**」 = 1999-2002 那代
