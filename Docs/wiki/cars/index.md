# 🚗 車輛介紹

> **比喻**：車就像**人**——有**體型**（重量分配）、**心臟**（引擎）、**身分證**（chassis code）、**性格**（操控傾向）。
> 看懂這些「人物設定」，才知道一輛車**天生擅長什麼**、**不擅長什麼**。

單一車型分析、車種類型（肌肉車 / GT / 拉力車）、車系比較；以及 QuickTune 體系的車型分類框架。

## 🩺 快查：你的車是哪種「人格」？

### 看驅動方式

| 驅動 | 比喻 | 操控傾向 | 線上競爭力 |
|------|------|---------|-----------|
| **FWD** | 前驅 = 自己拉著腳踏車前進 | 易**推頭**（understeer）| **A 級以下** |
| **RWD** | 後驅 = 屁股後面有人推 | 易**甩尾**（oversteer）| 全級距強 |
| **AWD** | 四驅 = 四隻手腳都在動 | 平衡、最穩、加速強 | 全級距強，越野必備 |

### 看引擎位置

| 位置 | 比喻 | 操控特性 |
|------|------|---------|
| **前置（Front）** | 重物在頭頂——多數市售車 | 重量壓前胎，**穩** |
| **中置（Mid）** | 重物背在腰上——超跑（NSX、Ferrari）| 重心集中，**靈活、難駕馭** |
| **後置（Rear）** | 重物背在屁股——Porsche 911 | 重在後，**特殊性格、極限快但難開** |

### 看引擎類型

| 類型 | 比喻 |
|------|------|
| **V** 形（V6、V8、V12）| 字母 V 的形狀 — 美系 V8、義系 V12 經典 |
| **Inline / Straight**（直列）| 全部一字排開 — 日系 I6（2JZ、RB26）、現代 I4 |
| **Flat / H / Boxer**（水平對置）| 拳擊手互打 — Subaru、Porsche 招牌 |
| **W** | 罕見的 W 形 — Bugatti 等 |
| **Rotary**（轉子）| 沒活塞，只有旋轉的「三角片」— Mazda 招牌 |

### 看 chassis code

聽到「**FC、FD、ND、E46、C8、AE86**」這類字母+數字組合 → 多半是 chassis code，指**特定世代版本**。詳見 [車型代號](./車型代號.md)。

## 🎨 改裝風格分類（Build 類型）

> 同一輛車可以變成完全不同性格：

| Build 類型 | 比喻 | 特性 |
|------------|------|------|
| **Meta build** | 競技選手——只看勝率 | 對某類比賽**最強**，常被認為**廉價、不真實** |
| **Purist build** | 古典派——保留原車味道 | 不換引擎、不換驅動、不裝 Forza 招牌前擾流 |
| **Power build** | 肌肉怪獸 | **全部 PI 砸馬力**、犧牲操控；適合 sprint |
| **Grip / Handling build** | 舞者 | **重抓地、操控好**，馬力較低；適合圈速 |

> 詳見 [汽車基礎術語](./汽車基礎術語.md)「Build 類型分類」。
